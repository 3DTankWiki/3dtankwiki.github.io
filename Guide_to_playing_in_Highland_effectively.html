关于高原

夏季版本 冬季版本
类别 详情
概览 高原是一张相当复杂的地图——它拥有不同的高度层，在某些地方非常狭窄，需要特定的技能才能游玩。这是一张中型地图，是掌握近距离防御和进攻战术的绝佳训练场所。它为快节奏的游戏体验提供了良好的环境，只有少数玩家注意到了这一点并善加利用。它也是一个非常受欢迎的跑酷场所。中短程炮塔在这里会如鱼得水，但由于两个基地之间空间广阔，像激光炮或镭射炮这样的远程炮塔有时也会出现。
游戏模式 死亡混战，团队对战，互夺军旗，控制点
规模 8 对 8，16 名玩家

战术

模式 策略
死亡混战

蹲守战术
只要你使用镭射炮、激光炮或马格南，在高原地图蹲守是一个不错的战术。你看，如果你使用的不是这些炮塔，那么比起狙击，你最好还是在中场游走；这主要是因为地图本身的大小所致。

例如，如果你想用雷暴炮或轰天炮之类的武器进行狙击，我建议你放弃你的位置，全力投入到中场战斗中。

此外，狙击需要一些基本的挑选目标意识，并确保他们在发挥战斗作用之前就被消灭。例如，如果你面临着在射击一个向你冲来的坦克和一个较弱但距离较远的坦克之间的两难选择，先射击那个较弱的。你将抢到击杀，降低那个人获胜的机会，同时消灭那个向你冲来的人。

至于道具，我建议使用伤害提升来增加你的伤害输出；尽管你应该保留一些护甲提升，以防整场战斗决定彻底终结你。

此外，你可以在高原地图中选择各种各样的蹲守点，无论是在蓝队基地后的房子后面，还是在红队基地的这个区域。

撕裂战术
基本上，如果你有不错的装备，例如离子炮 / 轰天炮 / 雷暴炮 / 火龙珠搭配维京 / 泰坦，你可以在地图中心与大多数敌人进行对抗。

无论是“嗑药战”还是“非嗑药战”，你都必须与他人保持一致——高效使用道具并配合掉落的补给箱，地图中间的维修工具，以及红队基地栅栏后掉落的伤害提升和护甲提升箱，根据此地图的团队模式，这是获得最大收益的关键。

如果你即将被摧毁，不要犹豫，离开“体育场”，在死亡并损失道具之前尽量多杀敌。不推荐给道具不足且装备较弱的玩家使用。

潜行与偷袭战术
这是一种战术，你需要用合适的装备预测你应该瞄准哪些敌人。一把激光炮 / 雷暴炮 / 滑膛炮，有时搭配冰风暴 / 火焰炮以及蜂王 / 黄蜂轻甲是此角色的完美组合。

你必须有敏捷的头脑和手指，对敌人装备的统计数据有深入了解。在敌人较少的区域潜行和奔跑，打击、奔跑、击杀，重复此过程。

建议猎杀装备较弱且生命值较低的敌人，观察哪个敌人正处于其他敌人的火力之下且即将死亡，保持距离并在适当的时机跳出来抢人头。结果取决于你使用此战术的熟练程度。

建议在战术之间进行切换，因为这完全取决于所面对的敌人。

全力以赴战术
全力以赴进攻是高原死亡混战中人们使用的最佳战术之一。基本上，它的工作原理是你选择一个伤害高但重装时间较慢的炮塔。

此类炮塔的典型例子是轰天炮、火箭炮和离子炮。只需抓一个重型底盘，例如泰坦，然后摧毁你路径上看到的每个人。

请注意，如果你想让这个战术奏效，你必须使用你的道具。在执行此战术时，你会面对很多人，所以不时更换位置也是一个战略建议。

例如，我在这里使用滑膛炮——利用它的颗粒射击墙对面的坦克。在此之后，我会迅速更换位置并继续我的狂欢。

团队对战

在团队对战中，最大的目标是在不被杀的情况下杀敌。这意味着你的团队中应该有足够的磁力炮来补充哪怕是最轻微的生命值损失。

大多数人倾向于在这种格式中全力以赴，试图尽可能早、尽可能快地结束游戏；但这并不总是奏效。

你需要三样基本东西来赢得高原团队对战：掩体、比敌人更大的伤害输出以及磁力炮。

虽然，和死亡混战一样，建议使用更高效的装备之一。这里最好的战术是包围并征服敌人的基地，并不断向他们开火。

除了建议的死亡混战装备外，一个好的团队玩家应该穿戴磁力炮 / 轰天炮 / 滑膛炮搭配维京 / 猎人中甲。正如所有优秀的团队一样，你的团队必须共同努力才能保证胜利。

悬崖战术
大多数人不知道这一点，但在红队基地的第一个悬崖上没有自毁区。

你可以通过独自驾驶黄蜂轻甲和雷暴炮 / 火龙珠 / 马格南到达悬崖上来利用这一事实，或者让团队中的某人以泰坦或猛犸象的身份把你送上去。

如果你能迅速安排一个设置，让你作为蓝队成员到达那里，你将拥有地图上最大的制高点——这意味着你的团队将毫无问题地接管另一个团队。

然而，上去只会给你一个掩体柱，所以强烈建议准备一些护甲提升和维修工具。

磁力炮蜂群战术
这个战术效果很好，很多使用它的团队大部分时间都能获胜。团队中的大多数人都会选择磁力炮。

这为两件事打开了空间：

一个重型底盘突然对敌方团队变得无敌，因为三到四个磁力炮同时在治疗他/她和彼此。这个战术可以消灭整个敌方团队。

整个团队变得致命——随着磁力炮不断治疗其他玩家，该团队的死亡损失大大降低，使他们更容易消灭对方团队中的人。

互夺军旗

防御 — 红方
在这一侧游玩提供了更好的防御。进入放置旗帜的红队基地区域只有 2 条路——穿过通道，以及从旗帜左侧的悬崖跳下，从红蓝旗帜线的视角来看。

栅栏、墙壁和许多其他物体非常有帮助，也提供了很好的蹲守点。例如；栅栏上的洞，特别是靠近小型箱状白色建筑左侧的一个，这也让防守者除了防守之外还能帮助他的团队。

如果进攻性防御更适合你，在红蓝基地区域周围驾驶并击倒进来的玩家，阻挡最危险的攻击者和旗帜持有者，并放置一些地雷来支持你的防御，这是你的完美解决方案。

这一侧的防守者应装备任何射程的炮塔，建议：镭射炮 / 火焰炮 / 离子炮 / 冰风暴，取决于他们选择的具体角色和位置，并搭配像猎人中甲、维京或猛犸象这样具有相当护甲的底盘。

支持者和进攻性防守者应选择进攻性组合，如雷暴炮 / 激光炮 / 火箭炮 / 雷暴炮 / 火龙珠或磁力炮，担任治疗者角色，并搭配维京 / 泰坦 / 蜂王。

防御 — 蓝方
在这一侧游玩提供了更多的蹲守点和更好的战场视野，但也为红方的攻击者提供了更多的路径和进入其基地的方式。

这一侧的防御可能比对手稍微困难一些，但在“足够好的位置”方面并不缺乏。就像另一侧一样，建议通过使用“蹲守防御”战术来对付敌人。

不要停下来等待敌方攻击者出现，在其他穿着近/中程武器的防守者的帮助下，拿一把远程武器，通过射击进入你视野的进攻性敌人来支持攻击者，同时如果严重的攻击发生在你方防御上，也要扮演防守者的原始角色，或者扮演普通防守者并利用该区域能提供的所有优势。

密切关注蓝队区域三个关键掉落区，并尝试控制它们，连同郊区和地图中间。

如果你认为你准备好在这一侧扮演更困难的角色并进行良好的防御，你可能会喜欢像冰风暴搭配黄蜂轻甲 / 猎人中甲、镭射炮-维京、离子炮-猛犸象、滑膛炮-猎人中甲以及类似的组合。

支持者和进攻性防守者可以选择任何带有稳定底盘的炮塔，如维京或猎人中甲。

这是一张建议位置和当前位置潜在视野的地图：防守者是较暗的点，而进攻性防守者/支持者是较浅的点。

进攻 — 红方
在红方进攻可能比在对面更难，为了表现出色并为你的团队带来胜利，攻击者必须比敌人的防御打得更努力。

虽然有可能，但如果进攻缺乏全力，这通常意味着敌人将开始更猛烈地进攻，而你团队中的玩家可能不得不改变他们的角色，这会导致失败。

为了避免这一切，解决方案来自规划进攻、路线和正确的时机。在进攻性防守者和支持者的支持下，攻击者可以开始施加影响，以安全夺取旗帜。

一个非常适合的攻击者应该拥有一个快速且健康的底盘，并配备中远程炮塔。

进攻 — 蓝方
在蓝方进攻可能比在对面更容易，但从敌人的基地活着带回旗帜可能会困难得多，如果他们的防御非常好，几乎是不可能的。

然而，如果在主攻之前投入了正确的力量和多次进攻浪潮，并在团队和援助的大力帮助下，旗帜是可以被夺取和捕获的。使用之前提到的团队对战战术在这样做时可能会很有用。

攻击者应该比配备中程炮塔的敏捷底盘拥有更多的生命值。

攻击者的不错组合：离子炮或火龙珠搭配维京、激光炮-黄蜂轻甲、滑膛炮搭配猎人中甲或蜂王。

进攻性支持者的不错组合：磁力炮或雷暴炮搭配蜂王或维京、激光炮或火箭炮搭配维京。

这是一张用于规划进攻的建议路线地图：深色部分 = 在障碍物后面/下面，带字母的点 = 攻击者开始的地方，不一定；B 可以以任何其余点结束，蓝方，A 应该使用通往没有字母的点的路线结束，这是相同的原始终点。

控制点

为了在高原提供的最后一个团队模式中获得胜利，这也是通常游戏玩法最残酷的模式，团队必须持有下图所示位置的三个点。

谈到战术，应该包括一些。通常建议占领离基地最近的点，所以如果你的团队这样做并没有错。

团队中的一名玩家，离点最近的那个人，应该占领你基地中的点并留在那里防守它——B 和 C 两个点都位于很好的蹲守位置，并且靠近道具掉落区，或者应该占领它并帮助团队征服其余部分。

如果你计划征服其他两个点，帮助是不可避免的。

B 和 C 点应始终由各自的团队首先观察和占领，团队应开始争夺位于地图中间的 A 点。

虽然，当其余人在战斗时，来自同一团队的一两名玩家应该互相支持以占领敌方基地区域的点。

高原控制点战斗的好玩家可以穿戴任何射程的炮塔，取决于底盘，底盘应该足够健康和稳定，以便尽可能长时间地留在周围，以保持点为你团队的颜色。

在众多组合中，这里列出了一些适合此类玩家的好组合：离子炮搭配猛犸象或维京、雷暴炮-独裁者、激光炮或磁力炮搭配维京、马格南搭配任何中型底盘。

火焰炮和冰风暴都非常有用，因为它们可以在极短的时间内消灭一堆坦克；并且可以很好地对抗敌人的维修工具。这将使其他人更容易占领点。

此外，你需要等待。耐心是赢得控制点游戏的必要条件。大多数人确实会独自全力以赴，而这只会使他们的团队变弱。
