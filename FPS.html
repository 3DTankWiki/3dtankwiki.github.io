什么是帧数

帧数（FPS，每秒帧数）是指屏幕上帧（图像）变化的频率。

换句话说，就是游戏画面每秒刷新的次数。帧数越高，你的游戏体验就越好（越流畅）。

你可以在屏幕右上角看到当前的帧数：

有些玩家认为帧数取决于互联网连接速度或服务器负载，但事实并非如此。该数值主要取决于你电脑的性能。

如果你的帧数较低，请按照以下说明操作。

如何提高帧数？ - 一般准则

首先，检查你的设置。确保所有图形效果都已禁用。

要查看设置，请点击左下角的“齿轮”按钮。如果你在战斗中，可以在暂停菜单中找到设置按钮，你可以通过按 Escape 或 P 键打开它。

  • 如果你使用笔记本电脑，请确保在操作系统的电源选项中激活“卓越”（或“高性能”）模式。
  • 将浏览器更新到最新版本。通常可以从浏览器菜单的“关于”部分进行操作。
  • 前往 https://webglreport.com/?v=2，找到“Unmasked Renderer”值，并确保在此参数中选择了正确的显卡。
    • 如果你看到“Google Swift”而不是你的显卡，请更新显卡驱动程序。
    • 如果你有一台带有两张显卡的笔记本电脑，并且网站显示的是集成显卡（例如 Intel HD Graphics）而不是独立显卡，请按照本页下方“带有 2 张显卡的笔记本电脑”标题下的说明进行操作。
  • 下载并安装最新的显卡驱动程序。以下是一些常见品牌的链接：NVIDIA, AMD / ATI, Intel

重要！如果你有笔记本电脑或一体机 PC，你需要从制造商的网站下载适用于你型号的驱动程序。

  • 关闭除游戏标签页之外的所有浏览器标签页。
  • 关闭所有其他程序以释放一些系统资源。
  • 尝试重启你的电脑。
  • 尝试使用以下浏览器之一：Google Chrome, Opera, Yandex, Vivaldi。

要在 Windows 上查找显卡名称，请按 Win+R。在弹出的窗口中，输入“dxdiag”（不带引号）并按 Enter。

应该会弹出一个新窗口。点击“显示”选项卡。

要在 macOS 上查找显卡名称，请点击左上角的 Apple 图标，然后点击“关于本机”。

显卡设置

对于 NVIDIA 显卡：

  1. 右键点击桌面并选择“NVIDIA 控制面板” → “通过预览调整图像设置”。
  2. 选择“使用我的优先选择，侧重于：”并将滑块一直向左移动到“性能”。
  3. 点击“应用”。


对于 ATI/AMD 显卡：

  1. 右键点击桌面并选择“AMD Catalyst Control Center” → “默认参数” → “应用用户首选项”。
  2. 选择“性能”。
  3. 选择“MipMap 细节级别”选项卡 → 将滑块向左移动 → “性能”。
  4. 点击“应用” → “确定”。

如果上述方法无效，在 HTML5 上提高帧数的另一种方法

许多玩家报告 HTML5 版本游戏帧率极低且延迟很高，这可能是由多种原因造成的。

  • 你的电脑性能不够强大，目前 HTML5 比 Flash 要求更高，尽管开发人员正在对其进行优化。
  • 更常见的是，你在浏览器中禁用了硬件加速。

HWA（我们现在这样称呼它）将运行游戏的工作从 CPU 卸载到 GPU，GPU 是专为运行游戏而设计的。这释放了你的 CPU 以执行更重要的任务。

首先，你需要检查 HWA 是开启还是关闭。这适用于 CHROME 和 OPERA，其他 Chromium 浏览器未经测试，但步骤应该类似或相同。（HTML5 仅适用于 Chromium 浏览器）

在搜索栏中输入：

chrome://gpu/

这应该会打开一个看起来像这样的页面：

如果你看到所有这些选项都显示为绿色的“Hardware accelerated”（硬件加速），那么你已经启用了 HWA，你的低性能是由其他原因引起的。

如果你看到“Software only, hardware acceleration unavailable/disabled”（仅软件，硬件加速不可用/已禁用），那么你需要进入浏览器设置：

在搜索栏中输入：

chrome://settings

按照下面的图片操作：滚动到底部 > 高级 > 系统 > 使用硬件加速（如果可用）

    • 滚动到底部


    • > 高级


    • > 系统 > 使用硬件加速（如果可用）


启用该选项并重启浏览器，你现在应该在 GPU 页面上看到“Hardware accelerated”（硬件加速），并体验到更好的性能。
