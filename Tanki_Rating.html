3D 坦克排名门户

TankiRating 是一个包含“3D 坦克”游戏统计数据的门户网站。该网站与官方资源无关，仅是一个社区项目。

在门户网站上，您可以看到在网站上注册的玩家的游戏统计数据。与官方排名网站不同，该门户网站以“动态”方式提供数据，也就是说，可以查看例如游戏时间的图表，这将清晰明了地显示某个坦克手在某一天（周、月）填充了多少小时、经验或赚取了多少水晶。除了模拟时间外，还监控了许多其他参数，但写关于它们的内容很无聊——只需转到玩家页面并亲自研究功能！

如何注册以将您的账户资料添加到门户网站

通过搜索您的昵称来检查您的个人资料——如果您已经在门户网站上注册，则无需执行任何操作。如果没有，请从您的游戏账户在俄罗斯论坛上向名为 sudo 的玩家发送私信。消息的主题和内容并不重要。

一天之内，该账户将被添加到网站。如果没有发生这种情况，请向玩家 Fizzika 发送私信

数据收集

活动数据的收集从在门户网站注册的那一刻开始。注册后，您将收到一个“空”的个人资料，它会随着时间的推移开始更新。随着时间的推移，个人资料将填充数据，这些数据将对门户网站的所有访问者可见。跟踪使用来自官方排名网站的开放 API 进行。该服务不需要您游戏账户的任何密码，使用它是绝对安全的。

在线中心部分

该网站为最新访客提供了在线时间表。您可以通过点击页面标题中的“在线中心”链接到达那里。除了 CCU 时间表外，还有一个 PCU 时间表。以后还会有新的功能和指标。
