游戏无法加载

如果进入游戏后屏幕上几乎立即出现错误，可能是其他人正在登录您的账户。请更改您的电子邮箱和密码。如果无法登录您的账户，请联系技术支持。

其他解决方案：

  1. 尝试更换浏览器。浏览器始终可以在官方网站上下载；最适合游戏的浏览器是 Chrome、Yandex.Browser、Firefox、Opera 和 Opera GX。
  2. 您的互联网连接可能太慢 - 请重新连接（重启路由器或拔掉电缆再插上）。停止所有下载，关闭所有种子客户端。
  3. 清理您的浏览器或客户端缓存。
  4. 尝试按照加速游戏加载的说明进行操作。

如果游戏无法启动怎么办？

新版浏览器 3D 坦克仅支持现代浏览器。其中包括最新版本的 Chrome、Mozilla Firefox、Opera 和 Yandex.Browser。我们建议使用 Chrome。

游戏无法在 Internet Explorer 中运行。

此外，您的计算机上必须下载最新的稳定版显卡驱动程序。这一点很重要，因为浏览器、驱动程序版本和显卡的某些组合可能会导致兼容性问题，而这些问题会在新版本的软件中得到修复。

  1. 将您的浏览器更新到可用的最新版本（通常可以通过浏览器菜单 → 关于程序进行更新）。
  2. 下载并更新您的显卡的最新驱动程序版本。

NVIDIA AMD/ATI Intel

要在 Windows 操作系统的计算机上检查您的显卡型号，请按 WIN + R 组合键，输入 dxdiag 并按 ENTER。在出现的窗口中，选择“显示”部分。

请注意！如果您的计算机是笔记本电脑或一体机，驱动程序应从您计算机型号制造商的网站下载。


对于拥有集成显卡和独立显卡的计算机用户：

  1. 打开显卡控制面板（通常可以通过右键点击桌面访问的菜单中找到相应选项），并将您的浏览器设置为使用独立（高性能）显卡。
  2. 访问网站 https://webglreport.com/?v=2 并确保：
    • WebGL 2 已受支持并被识别（您会在绿色背景上看到“✓ This browser supports WebGL 2”标签）；
    • 识别出的是独立显卡（NVIDIA Geforce 或 AMD/ATI Radeon），而不是集成显卡（Intel HD, AMD APU）。


对于 Chrome, Opera, Yandex：

  1. 右键点击浏览器快捷方式并选择“属性”选项。快捷方式属性窗口将打开。
  2. 点击“快捷方式”部分中的“目标”行，将光标放在行尾，并添加一个空格。
  3. 在光标位置插入以下文本： --disable-gpu-driver-bug-workarounds --ignore-gpu-blacklist --enable-d3d11
  1. 按“确定”保存更改。
  2. 如果浏览器已启动，请将其关闭并重新启动。关闭浏览器窗口后，打开 Windows 任务管理器，确保没有名为 chrome.exe 的进程正在运行（或者，如果有此类进程，请手动停止它们）。


对于 Firefox：

  1. 前往 about:config（将其粘贴到浏览器地址栏并按 ENTER）。浏览器配置页面将打开。
  2. 使用搜索栏查找 webgl.angle.force-d3d11 属性。
  3. 将该属性的值设置为 true。
  4. 重启浏览器。


注意
请注意！按照上述说明操作可能会导致浏览器和整个系统运行不稳定。在这种情况下，建议撤销更改并恢复默认设置。
