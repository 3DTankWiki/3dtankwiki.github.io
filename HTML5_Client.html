游戏未加载

如果在进入游戏后几乎立即在屏幕上出现错误，可能有人正在登录你的账户账号. 更改你的电子邮件和密码。如果无法登录你的账号，请联系技术支持.

其他解决方案：

  1. 尝试更换浏览器。浏览器总是可以在官方网站上下载；最适合游戏的浏览器是谷歌浏览器,Yandex 浏览器,火狐,歌剧，和歌剧 GX.
  2. 你的网络连接可能太慢——重新连接（重启路由器或拔掉并重新插入网线）。停止所有下载，关闭所有种子客户端。
  3. 清除你的浏览器或客户端缓存。
  4. 尝试使用该指令加速游戏的加载。

如果游戏无法启动该怎么办？

3D坦克的新浏览器版本仅支持现代浏览器。这些包括最新版本的Chrome、Mozilla Firefox、Opera和Yandex.Browser。我们推荐使用Chrome。

游戏不会起作用在 Internet Explorer 中。

除此之外，必须在你的电脑上下载最新的稳定图形卡驱动程序。这很重要，因为某些浏览器、驱动程序版本和图形卡的组合可能会导致兼容性问题，而这些问题在新版本的软件中已被修复。

  1. 将您的浏览器更新到可用的最新版本（通常可以通过浏览器菜单 → 关于程序来完成）。
  2. 下载并更新适用于您的显卡的最新版本驱动程序。

英伟达 AMD/ATI 英特尔

要在 Windows 操作系统的计算机上检查你的显卡型号，请按WIN+R按键组合，类型dxdiag并按ENTER在出现的窗口中，选择“显示”部分。

注意！如果您的电脑是笔记本电脑或一体机，驱动器应从您电脑型号制造商的网站下载。


对于拥有集成和独立显卡的电脑的用户：

  1. 打开图形控制面板（通常可以通过右键单击桌面访问的菜单中找到相应的选项）并设置专用（高性能）图形卡用于你的浏览器。
  2. 前往该网站https://webglreport.com/?v=2并确保：
    • 支持并识别 WebGL 2（您将在绿色背景上看到标签“✓ 此浏览器支持 WebGL 2”）；
    • 具体来说，是专用图形卡（NVIDIA Geforce 或 AMD/ATI Radeon）被识别，而不是集成显卡（Intel HD，AMD APU）。


为了谷歌浏览器，歌剧，Yandex：

  1. 右键点击浏览器快捷方式，然后选择“属性”选项。快捷方式属性窗口将会打开。
  2. 点击“快捷方式”部分的“目标”一行，将光标置于行末，然后添加一个宇宙。
  3. 在光标的职务处插入以下文本: --disable-gpu-驱动器-bug-workarounds --ignore-gpu-blacklist --enable-d3d11
  1. 按“确定”保存更改。
  2. 如果浏览器已经启动，请关闭浏览器，然后重新启动它。关闭浏览器窗口后，打开 Windows 任务管理器，确保没有名为 chrome.exe 的进程正在运行（或者，如果有这样的进程，请手动停止它们）。


为了火狐浏览器：

  1. 去关于:配置（将此粘贴到浏览器地址栏并按ENTER). 浏览器配置页面将会打开。
  2. 使用搜索栏查找 webgl.angle.force-d3d11 属性。
  3. 将属性的值设置为真.
  4. 重启浏览器。


注意
注意！按照上述说明操作可能导致浏览器和整个系统的不稳定工作。在这种情况下，建议撤销更改并恢复默认设置。
