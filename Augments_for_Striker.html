装备改造用于火箭炮

肾上腺素 245 000

当坦克受到严重损坏时，增加炮塔伤害。为增强伤害调制器添加一个备用电池，只在坦克底盘严重损坏的极端情况下激活。从两个能源源供电给伤害调制器是不安全的——可能会爆炸。要求仅在严重损坏时激活它，很可能是为了避免保修索赔。
优点
  • 普通和暴击伤害:+25%
缺点
  • 伤害加成仅在生命值为时激活≤25%
  • 当坦克有时，效果被禁用。冷冻状态效果.


远程火箭炸药 245 000

火箭配备了无接触触发器和无线发射器。双击发射按钮时，火箭仍在空中，将立即引爆它。在射击单发时要小心！
优点
  • 火箭将在双击发射按钮后爆炸
  • 最低火箭速度: 20米/秒
缺点
  • 火箭加速时间：1.5秒
  • 如果不小心，有自我伤害的可能性
  • 水平自动瞄准角度：±0°


导弹发射器 «猎人中甲» 245 000

为高精准射击的粉丝进行的改装。在瞄准时提供几乎瞬时的目标获取，但每次发射只有一枚火箭。
优点
  • 瞄准时间: -75%
  • 齐射炮弹重装时间: -40%
缺点
  • 火箭每齐射: 1
  • 瞄准恢复时间：-30%


导弹发射器 «旋风» 245 000

针对猎杀重型坦克的改装。每次发射的火箭数量翻倍，但瞄准目标的获取时间也增加了。
优点
  • 火箭发射次数: 8
  • 暂停齐射的火箭：-40%
  • 瞄准恢复时间：+50%
缺点
  • 瞄准时间: +20%


燃烧导弹 4 990

暴击伤害使目标在短时间内着火。一个带有特殊火箭转换器的改装，在暴击命中时启用。将标准导弹转换为燃烧导弹，让你将对手 engulf in flames，持续造成每秒伤害一段时间。
优点
  • 暴击击中会使所有受到伤害的敌方坦克的温度提高 +0.4 着火
  • 暴击命中几率: +10%
缺点
  • 暴击伤害: -50%


冷冻导弹 4 990

暴击伤害会使敌方坦克短时间内受到«冷冻»状态效果的影响。一种带有特殊火箭转换器的改装，可以在暴击命中时启用。将标准导弹转换为冷冻导弹。这可以让你在战场上减缓敌人的移动，并且几乎将他们的炮塔暂时焊接到底盘上。
优点
  • 暴击攻击会降低所有受损敌方坦克的温度1。 冷冻
  • 暴击命中几率: +10%
缺点
  • 暴击伤害: -50%


磁性导弹 4 990

暴击伤害将 «电磁脉冲» 状态效果施加到敌方坦克上，持续短时间。一个带有特殊火箭转换器的改装，在暴击时启用。将标准导弹转换为电磁导弹。导致敌方坦克的电子设备短暂关闭，使坦克的道具停用一段时间。
优点
  • 对敌人进行关键打击将会施加 电磁脉冲状态效果持续3秒。
  • 暴击命中几率: +10%
缺点
  • 暴击伤害: -50%


眩晕导弹 4 990

一个修改，用于使危险目标失去行动能力。任何受到暴击伤害的坦克将在一段时间内无法移动。
优点
  • 对敌人进行关键打击将会施加 眩晕状态效果持续1.5秒。
  • 暴击命中几率: +10%
缺点
  • 暴击伤害: -50%


穿甲导弹 4 990

一个为想要摧毁强大目标的赏金猎人中甲准备的装备改造。任何受到暴击伤害的坦克将在一段时间内无法使用护甲。
优点
  • 对敌人进行关键打击将会施加 护甲穿透状态效果对他们持续9秒。
  • 暴击命中几率: +10%
缺点
  • 暴击伤害: -50%


干扰导弹 4 990

伤害将干扰器状态效果施加给对手，持续短暂时间。
导弹装备有铝涂层的玻璃纤维。在成功命中后，会产生一小团云，能使红队的过速装置失效。
优点
  • 对敌人进行关键打击将会施加 干扰器状态效果持续5秒。
缺点
  • 暴击伤害: -50%


导弹发射器«轴»

为喜爱大爆炸的坦克手们准备的装备改造。强大的启动加速度伴随着巨大的责任！提升你选择目标和时机的能力，你的火箭将为战斗带来胜利。
优点
  • 伤害: +125%
  • 暴击伤害: +160%
  • 瞄准时间: -35%
  • 最低溅射伤害半径: +50%
  • 最高投射物速度: 350 米/秒
缺点
  • 投射物加速时间：+350%
  • 单发炮弹重装时间：+15%
  • 暂停齐射的火箭：+400%


导弹发射器 «串联»

遇到了一个经验丰富的对手，他能预测你每一步动作并躲避火箭？那就发射一枚快速追踪火箭！对手是不是太厉害了，以至于也躲过了这一发？那就同时发射两枚快速追踪火箭！让他们试着躲避这两发。
优点
  • 瞄准时间: -75%
  • 火箭角速度: +25%
  • 暴击命中几率: +25%
  • 暂停齐射的火箭：-100%
缺点
  • 齐射后的充能时间: +35%
  • 火箭每齐射: 2


导弹发射器«九头蛇»

一种快速发射的火箭发射器。齐射可以发射大量失去控制的火箭，其伤害减小。这些火箭曾经安装在攻击直升机上。由于缺乏瞄准系统以及较小的大小，可以将多个火箭塞入发射器中。齐射攻击的长持续时间允许在射击时切换红队，而不会中断齐射火箭的发射。坦克手本来就没有选择中断火箭发射的余地。
优点
  • 每个齐射的火箭数 = 20
  • 暂停齐射的火箭：-20%
  • 瞄准时间: -50%
  • 单发炮弹重装时间：-40%
缺点
  • 普通和暴击伤害: -50%
  • 齐射后的充能时间: +80%
  • 火箭的角速度 = 3°/秒


导弹发射器«浮士德»

导弹的射程已减小，但瞄准目标和重新装填所需的时间更少。就像舞台上的艺术家，这位拥有装备改造的玩家在为观众表演。没有中场休息，只有迅速展开的战场活动。然而，这位艺术家是个初学者，有时他们未能将自己的想法传达给观众。
优点
  • 炮弹重装时间: -50%
  • 瞄准时间: -50%
  • 暂停齐射的火箭：-50%
缺点
  • 射程 = 40米
  • 重新加载后齐射: +100%


真空导弹

目标离爆炸中心越远，受到的伤害就越多。再次按下发射按钮会立即在导弹仍在空中时引爆它。独特的远程和真空爆炸技术结合，使得快速飞行的导弹如果瞄准靠近目标，但不直接瞄准目标，将会更有效。爆炸的导弹产生的冲击波造成的伤害大于直接命中。
优点
  • 平均和最小溅射伤害：230%
  • 火箭将在双击发射按钮后爆炸
  • 最低火箭速度: 20米/秒
  • 水平自动瞄准角度：±0°
缺点
  • 伤害: -30%
  • 火箭加速时间：1.5秒
  • 如果不小心，有自我伤害的可能性
  • 暂停齐射火箭：+12%


导弹发射器 «流星»

增加对目标的伤害。溅射伤害大大减少，但冲击力增加。感受流星的力量！你的敌人几乎没有幸存的几率。然而，要利用如此强大的启动加速度，你必须小心瞄准。这是为真正的导弹专业人士设计的装备改造！
优点
  • 伤害: +25%
  • 冲击力: +50%
  • 最低火箭速度 = 90米/秒
  • 火箭齐射之间的暂停 = 50毫秒
缺点
  • 溅射半径 = 1米


导弹发射器 «铜头»

加速火箭齐射目标定位，但显著增加射击后的炮弹重装时间。启用火箭的无线引爆。
这个修改是在分析了无数火箭炮战斗场景后开发的。增强的炮塔装甲确保发射器保持满载火箭，从而实现快速齐射部署。然而，在齐射后重新装填新火箭所需的时间要长得多。
作为对一个机密请求的回应，发射器配备了一个无线电引爆防御模块，用于火箭的中途引爆。此功能的细节仍然是绝密的。
优点
  • 火箭将在双击发射按钮后爆炸
  • 瞄准时间: -62%
  • 暂停齐射的火箭：-20%
  • 最高暴击命中率 = 100%
  • 最低火箭速度: 20米/秒
缺点
  • 炮弹重装时间在齐射后: +111%
  • 火箭加速时间：1.5秒


露营者

当坦克的伤害非常少时，增加炮塔伤害。
一个备用电池用于增强伤害调制器，连接在主启动加速度源的并联电路中，通过一个AI控制的管理系统。这套系统可以安全地提升炮塔伤害，超过调制器的标准能力，不过前提是坦克的结构完整。如果AI检测到底盘被破坏，备用电池将立即关闭——这一切都是为了防止保修索赔，当然。
优点
  • 普通和暴击伤害增加:+100%
缺点
  • 伤害增益仅在您的当前生命值至少为最高生命值的65%时有效。
  • 当坦克有时，效果被禁用。 冷冻状态效果.


脉冲星


一种采用无法复制技术的装备改造。一个暴击击中会产生一个量子漩涡，同时施加几个效果，通常是负面的。科学家设法减少了暴击伤害，但这并不足以挽救局面——一旦被击中，敌人仍会受到严重的毁灭。这项技术的发明者在国际通缉名单上。
优点
  • 用暴击射击击中敌人将施加以下状态效果：
    • 干扰器5秒
    • 电磁脉冲: 3秒
    • 眩晕1.5秒
    • 护甲穿透9秒
缺点
  • 暴击几率: -9%
  • 暴击伤害: -90%


凤凰 5 000 000

只有那些像凤凰一样一次又一次从灰烬中崛起以实现目标的坦克手们，才值得拥有这种武器。一种秘密装备改造。使用火箭炮炮塔获得 5,000,000 名声点数即可得到它。
优点
  • 伤害: +15%

  • 暴击几率: +30%
  • 垂直自动瞄准：+15%
