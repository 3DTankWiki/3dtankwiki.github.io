语音通信

另见:社交网络

语音通信允许玩家在游戏过程中实时互相交流。这在战斗、军团活动或电子竞技活动中尤其有用，因为玩家可以在不暂停打字聊天的情况下进行协调。3D坦克不包含内置语音聊天, 玩家通常依赖第三方应用程序。

以下是最常用的语音通信程序以及有关其功能和安装的最新信息。

Discord

Discord是一个免费的语音、视频和文本通信平台，可在桌面、移动设备和浏览器上使用。目前，它是游戏玩家和电子竞技团队中最受欢迎的通信工具。
功能
  • 低 CPU 和内存使用率。
  • 易于使用的界面，具有可自定义的权限。
  • 通过 Discord 的全球基础设施进行 DDoS 和 IP 防护。
  • 高质量语音频道、视频通话和屏幕共享。
  • 每用户音量调节和山顶城堡抑制功能。
  • 免费服务器创建，拥有无限的文字/语音频道。
  • 现代功能，如舞台频道、线程和服务器社区。

系统要求 (2025)

  • Windows 10及以上，macOS 10.13及以上，Linux（.deb / .rpm），iOS 12及以上，Android 7及以上。
  • 内存：最低 512 MB。
  • 磁盘空间: 约 250–300 MB 免费。
  • 任何标准声卡或耳机。

安装与使用

  1. 下载应用或使用网页版官方网站.
  2. 创建一个账号，然后加入已有的服务器或创建自己的服务器。

TeamSpeak

 TeamSpeak是一款长期存在的 VoIP 应用程序，广泛用于竞技游戏，以其稳定性、低延迟和先进的权限系统而闻名。
功能
  • 可选的自托管服务器（付费或自运行），包括细粒度权限控制。
  • 高度可配置的音频设置、音量等级以及麦克风激活模式。
  • 服务器文本聊天、二等兵消息传递以及基于频道的组织。
  • 能够在本地记录对话。
  • 非常低延迟且高性能的语音传输。

系统要求 (2025)

  • Windows 10 及以上版本，macOS 10.14 及以上版本，使用现代 glibc 的 Linux 发行版。
  • 支持 SSE2 的处理器。
  • 内存：512 MB以上。
  • 磁盘宇宙：约100 MB 免费。
  • 耳机或麦克风。

使用方法

  1. 从...下载并安装客户端官方网站.
  2. 启动 TeamSpeak。
  3. 可选：配置权限、麦克风和音频设置（工具 → 选项）。
  4. 打开“连接” → “连接”（或按Ctrl+S) 加入一个服务器。
  5. 输入服务器地址、昵称和密码（如果需要）。服务器详情通常由团队队长或管理员提供。
  6. 点击“连接”加入语音频道。

有趣的事实

  • 2019年，开发者曾透露一个新的3D坦克模式，其中一个玩家控制底盘，另一个玩家控制炮塔，需要语音聊天来进行协调。后来这被揭示为愚人节玩笑。
