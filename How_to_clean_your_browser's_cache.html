手动清理浏览器缓存

Google Chrome

  1. 点击浏览器工具栏上的 Chrome 菜单。
  2. 选择工具。
  3. 选择清除浏览数据。
  4. 勾选除“清除保存的密码”之外的所有框
  5. 使用顶部的菜单选择要删除的数据量。选择“时间不限”以删除所有内容。

6. 点击清除浏览数据。


Opera

  1. 在“工具”菜单上，点击删除私人数据…
  2. 在“删除私人数据”屏幕上，点击详细信息。

注意：在 Opera 10.0 及更高版本中，该按钮在左侧标记为“详细选项”。

  1. 勾选“删除整个缓存”框。然后点击删除。


Firefox

  1. 在 Firefox 窗口顶部，点击 Firefox 按钮，然后选择选项
  2. 选择高级面板。
  3. 点击网络选项卡。
  4. 在缓存的 Web 内容部分，点击立即清理。
  5. 点击确定关闭选项窗口。


Safari

Safari for Mac OS X

  1. 打开浏览器后，点击 Safari 菜单并选择清空缓存…
  2. 点击清空。

Safari for Windows

  1. 打开浏览器后，点击右上角的齿轮图标。
  2. 选择“重置 Safari…”这将提示打开一个屏幕。
  3. 选择提示底部的“删除所有网站数据”。勾选或取消勾选你想要重置的任何其他类别。
  4. 点击“重置”。


Microsoft Edge

  1. 打开 Microsoft Edge。
  2. 点击你的浏览器设置（可以通过浏览器右上角的三点图标看到）
  3. 点击“设置”按钮。
  4. 在搜索栏上，搜索单词“缓存”。
  5. 在“立即清除浏览数据”旁边，点击“选择要清除的内容”。
  6. 选择“缓存的图像和文件”以及“Cookie 和其他网站数据”选项。
  7. 点击“立即清除”。


使用 CCleaner 软件清理浏览器缓存和 Cookie

CCleaner 是一个可以做很多事情的程序，包括清理浏览器的缓存和 Cookie。 该过程只需几秒钟，你不需要进入浏览器的设置。你可以在此下载 CCleaner。 安装完成后，打开程序并点击应用程序选项卡。在那里找到你的浏览器并勾选 Internet 缓存和 Cookie 框，点击“运行清理”。

注意
清理缓存也会清理你的玩家忽略列表以及登录详细信息。
