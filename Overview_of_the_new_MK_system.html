新MK系统概述

see also:Mk转换,炮塔,底盘,防御模块和套件

在更新#586中，3D坦克中的修改等级工作方式已完全重做。进行了许多更改，包括将修改的缩写从“M”更改为“Mk”，平衡不同等级设备的成本，以及将修改的总数从四个更改为七个。

This page summarizes the changes in prices, 等级, and appearances for the new modification 等级. An even more detailed explanation can be found on the论坛 topic.

Changes to appearance

Prior to this update, each modification 等级 had its own unique "皮肤" (appearance). This changed with the Mk update. Two "Mk" 等级 will share the appearance of one "M" 等级, as illustrated in the table below.

Color code

  • 红色– a red number indicates that the price has been increased.
  • 白色– 一个白色数字表示价格保持大致不变。
  • Green– a green number indicates that the price has been decreased.
