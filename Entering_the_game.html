Linux 用户的独立安装指南

适用于 Linux 用户

1. 首先，如果你使用的是 Linux，你需要安装Nativefier你还需要 Node.js 和 npm。

2. 打开你的终端，导航到你想要安装它的地方，然后运行命令："sudo npm install -g nativefier"。如果你使用的是 Arch Linux，首先输入 "sudo pacman -S yay"，然后运行 "yay -S nativefier"。

3. 现在，在你的终端中运行此命令："nativefier"https://tankionline.com/战斗-en.html这将使 Nativefier 在你当前的目录中创建一个新文件夹。

4. 进入 Nativefier 创建的新文件夹，运行名为“TankiOnline”的可执行文件以启动游戏。

命令：cd TankiOnline-linux-x64 && ./TankiOnline
在哪里获取游戏
  • 适用于 Mac：Mac OS X (.dmg) 下载
  • 适用于 Windows：Windows 操作系统（.exe）下载


2.安装游戏！在 Windows 上，只需双击安装包并关注屏幕上的简单步骤。

Mac 玩家快讯！
Mac 用户将直接下载客户端程序，因此你不需要单独安装！

3. 一旦客户端设置完成，你可以轻松地从开始菜单启动它，或者只需搜索“TankiOnline”。

下载链接
  • 适用于 Mac OS X 的下载 (.dmg)

在任天堂Switch上进入3D坦克

快速介绍

这任天堂Switch真的无所不能——你可以在家将它连接到底座进行大屏游戏，或者随时随地拿起它玩。现在，3D坦克已经登陆Switch，所以那些快速坦克战斗会跟随你去任何地方。

描述

任天堂的Switch注重灵活性。可以在电视上玩，也可以放在桌子上，或者随身携带。3D坦克完美契合，无论你是在沙发上躺着还是在公交上，都可以立即投入坦克战斗。游戏已经为Switch进行了优化，所以操作流畅，无论使用Joy-Con还是Pro手柄，控制都非常舒适，并且可以连接你现有的设备。3D坦克账号很容易。

发布日期：2025年3月6日。起初，游戏上线时，你无法使用现有的账号。但好消息！在2025年10月9日的更新之后，玩家现在可以完全访问他们在PC或安卓设备上使用的主要账号。

如何跳入游戏

准备好展开行动了吗？以下是获取方式3D坦克在你的Switch上：


  1. 下载游戏— 前往任天堂电子商店直接从你的Switch上。寻找“3D坦克”并下载官方游戏。你也可以在美国商店中找到它，使用这个链接.
    提醒一下：任天堂eShop™的俄罗斯版本有一些限制。如果你的地区设置为俄罗斯，你可能无法下载新游戏。要获取3D坦克，请确保你的地区设置为其他国家。
  2. 启动游戏— 找到3D坦克在主屏幕上点击图标以启动。
  3. 登录或创建账户— 如果你已经是一个3D坦克玩家，你可以使用您常用的凭据登录这让你可以非常轻松地立即回到Switch上的游戏中。刚接触这款游戏？没问题！你可以直接从 Switch 界面创建一个账号开始起来非常轻松！
  4. 开始游戏——一旦你登录，游戏就会连接到服务器。你已经可以开始跳入战斗，按照自己的喜好装备你的坦克，并探索你在其他平台熟悉并喜爱的所有精彩游戏模式！


利与弊

随时随地玩- 使用掌上模式，你几乎可以在生活的任何地方玩3D坦克。
进步与你同行- 你所有的进度、购买和统计都会与你的 3D坦克账号同步，因此无论你是在 Switch、PC 还是 Android 上，一切都会保持一致。
平滑控制- Joy-Con 和 Pro 手柄都非常好用，让你专注于游戏动作。
图形调整- 为了在 Switch 上保持运行顺畅，一些图形设置相比 PC 被调低了一些。
简化聊天- 游戏内的聊天更简化，如果你习惯了电脑版，可能会觉得有些不同。
