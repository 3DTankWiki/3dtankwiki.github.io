装备改造

主要文章： 装备改造

冰风暴的装备改造


肾上腺素 245 000

当坦克受到严重损坏时增加炮塔伤害。 为增强型伤害调节器添加了一个备用电池，仅在坦克底盘受到严重损坏的极端情况下激活。从两个能源为伤害调节器供电是不安全的——它可能会爆炸。仅在严重损坏时激活的要求可能是为了避免保修索赔。
优点
  • 普通伤害和暴击伤害：+25%
缺点
  • 伤害加成仅在生命值 ≤35% 时激活
  • 当坦克具有冷冻状态效果时，效果被禁用。


高压泵 245 000

改善了中距离目标的命中距离：氟利昂流射程稍远，最大命中距离增加。需要更精确的瞄准。
优点
  • 最小伤害射程：+50%
  • 最大伤害射程：+50%
缺点
  • 锥角：-75%

腐蚀混合物 245 000

将冷却剂换成酸性混合物，能在几秒钟内腐蚀最坚固的装甲。无冷冻效果。
优点
  • 伤害：+10%
  • 暴击伤害：+10%
缺点
  • 移除冷冻效果

震荡冻结 245 000

快速有效地将坦克变成坚固且无法移动的冰块。由于炮塔内部空间有限，必须移除一些战斗机制，导致伤害降低。
优点
  • 冷冻率：-1.00 / 刻度
缺点
  • 伤害：-10%
  • 移除暴击

磁性混合物 4 990

在冷却剂中添加磁性混合物。混合物的临界浓度可能会导致目标的道具被禁用，但暴击伤害和冷冻速度减半。
优点

  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%
  • 冷冻率：-70%

麻痹混合物 4 990

在冷却剂中添加麻痹混合物。混合物的临界浓度可能会导致目标的所有系统瘫痪，但暴击伤害和冷冻速度减半。
优点
  • 暴击 眩晕敌人 0.4 秒
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%
  • 冷冻率：-70%

有毒混合物 4 990

将冷却剂换成有毒混合物。混合物的临界浓度可能会导致目标在短时间内无法使用护甲，但暴击伤害和冷冻率降低。
优点
  • 暴击命中敌人将对其施加 穿甲状态效果，持续 1.5 秒
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%
  • 冷冻率：-70%

干扰混合物 4 990

在冷却剂中添加静电混合物。混合物的临界浓度可能会导致目标在短时间内无法激活过速装置或使用无人机，但暴击伤害和冷冻率降低。
优点
  • 暴击命中敌人会激活 干扰器状态效果，持续 3 秒
缺点
  • 暴击伤害：-50%
  • 冷冻率：-70%

致命混合物

暴击几率增加。冷冻能力相应降低。 在冷冻剂中添加致命混合物。在组装该装备改造时，工程师在混合比例上犯了一个错误，结果导致暴击伤害发生的频率远高于开发此修改时的计划。但也有反面影响——制冷剂的比例急剧下降。
优点
  • 暴击几率：+500%
缺点
  • 每刻度冷冻：-50%

稳定混合物

冷冻效果降低，但攻击射程和能量储备显著增加。 此装备改造将您的炮塔变成了一种战术控制工具。通过削弱冷冻效果，它为您提供了更远的射程和更多的能量，使您能够轻松地将敌人保持在安全距离。有了它，伤害在长距离内保持稳定，您的战斗效能显著提高。
优点
  • 低伤害射程：+50%
  • 全伤害射程：+100%
  • 能量消耗：-50%
缺点
  • 冷冻率：-50%

露营者

当坦克受损较小时增加炮塔伤害。
为增强型伤害调节器配备了一个备用电池，通过人工智能控制的管理系统与主电源并联。该系统安全地将炮塔伤害提升至调节器标准能力之外，但前提是坦克的结构完整性保持不变。如果人工智能检测到底盘已受损，备用电池将立即关闭——当然，这一切都是为了避免保修索赔。
优点
  • 普通伤害和暴击伤害增加：+90%
缺点
  • 伤害加成仅在您拥有至少 50% 最大生命值时激活。
  • 当坦克具有 冷冻状态效果时，效果被禁用。

脉冲星


一种采用不可复制技术制造的装备改造。暴击会产生一个量子涡旋，同时施加多种效果，通常是负面效果。科学家们设法降低了暴击伤害，但这不足以挽救局面——一旦被击中，敌人仍会惨败。该技术的发明者在国际通缉名单上。
优点
  • 暴击命中敌人将施加以下状态效果：
    • 干扰器：3 秒
    • 电磁脉冲：1 秒
    • 眩晕：0.4 秒
    • 穿甲：1.5 秒
缺点
  • 暴击几率：-30%
  • 暴击伤害：-90%

卓越


增强关键炮塔或底盘参数，而不引入新功能。
这种高级装备改造是突破性研究计划的成果，将钻石晶体集成到炮塔设计中。钻石涂层的枪管衬里增加了射击时的操作压力。钻石轴承提高了炮塔的旋转效率。钻石半导体增强了瞄准系统的计算性能。唯一的缺点？其惊人的工艺隐藏在炮塔内部，无法从外部看到。
优点
  • 伤害：+30%
  • 转向速度：+15%
  • 转向加速度：+15%

特性表
  • 伤害 (hp/秒) — 每秒对敌方坦克造成的伤害量。
  • 伤害 (hp/刻度) — 每个刻度（1/4 秒）对敌方坦克造成的伤害量。
  • 暴击伤害 (hp/秒) — 受到暴击命中时对敌方坦克造成的伤害。
  • 暴击伤害几率 (%) — 可累积的暴击射击最高几率。

  • 旋转加速度 (度/秒²) — 炮塔达到其最大旋转速度的加速度。
  • 最大伤害射程 (m) — 炮塔造成全部伤害的距离。如果距离大于最大伤害射程，伤害会不断减少，直到坦克通过最小伤害射程。
  • 最小伤害射程 (m) — 炮塔伤害达到零并停止减少的距离。
  • 弱伤害 (%) — 在射程和锥角内对敌人造成的最小伤害。如果距离大于最大伤害射程，伤害会不断减少，直到坦克通过弱伤害射程。
  • 重装时间 (秒) — 如果冰风暴的能量已完全耗尽，补充其能量所需的总时间（秒）。
  • 能量总量 — 可用于射击的总能量。
  • 能量消耗 (eng/秒) — 如果连续使用冰风暴，每秒使用的能量量。
  • 装载 (ms) — 不使用武器时每秒再生的能量量。
  • 锥角 (度) — 霜冻从炮塔尖端散开的圆锥形向外范围（度）。在此圆锥体之外的对手无法受到伤害。
  • 冷冻率 (temp/秒) — 敌方底盘每秒降低的温度值。被冷冻时，坦克会稳步减速。
  • 炮塔冷冻等级 (%) — 被冷冻的敌方坦克炮塔在最低温度下移动的最大速度百分比。
  • 底盘冷冻等级 (%) — 被冷冻的敌方坦克底盘在最低温度下移动的最大速度百分比。
  • 锥角 (度) — 霜冻从炮塔尖端散开的圆锥形向外范围（度）。在此圆锥体之外的对手无法受到伤害。
  • 穿甲持续时间 - 造成暴击伤害刻度后，穿甲状态效果对敌人有效的秒数。
  • 干扰器持续时间 - 造成暴击伤害刻度后，干扰器状态效果对敌人有效的秒数。
  • 电磁脉冲持续时间 - 造成暴击伤害刻度后，电磁脉冲状态效果对敌人有效的秒数。
  • 眩晕持续时间 - 造成暴击伤害刻度后，眩晕状态效果对敌人有效的秒数。
