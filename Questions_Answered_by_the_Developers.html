Steam 版本的情况如何？ 伙计们，我们目前正在更新 Steam 版本，很快就会用游戏的 HTML5 版本替换平台上的 Flash 版本。在此举措之后，Flash 版本将最终被完全禁用。即使你找到了进入 Flash 版本的方法，在下一次重大更新后，它也将无法工作。这意味着我们将全面普及 HTML5 版本，这使我们能够专注于改进 HTML5 版本并为你带来更多酷炫的更新。你已经看到了其中之一——全新的界面。我们很快就会发布它。


你们会向匹配系统添加更多地图吗？ 我们希望你能理解，你连续 3 次进入同一张地图的事实可能与匹配系统的地图池大小无关。目前，我们已经有足够的地图，不需要向匹配系统的地图池中添加更多地图。相反，我们需要修复系统选择地图的方式。


你们会将地图的夜晚主题带回匹配系统吗？ 让我告诉你为什么我们现在没有它们：如果我没记错的话，我们前段时间在匹配系统中拥有过它们。当我们之前拥有它们时，我们只有游戏的 Flash 版本。Flash 版本有不同的照明系统。Flash 中的每个弹丸周围也有一个发光效果，可以照亮弹丸周围的物体。在白天，它几乎不可见。但在地图的夜晚主题中，你可以很容易地看到它，而且看起来非常华丽。你可能还记得夜间模式下的节日圣诞地图有多亮。在我们让 HTML5 版本中的照明工作方式与 Flash 版本中的工作方式相同（我们正计划这样做）之前，我们不会将地图的夜晚主题带回匹配系统，因为那样会太暗，很难玩。

激光炮的子弹穿透机制和 0 伤害错误何时修复？ 接下来是一个敏感问题。激光炮的子弹穿透机制何时修复？它已经乱七八糟很久了，当你尝试用激光炮射击超过 1 辆坦克时，子弹似乎在试图避免击中多人，而不是真正着陆。0 伤害错误也非常普遍，在许多情况下极其烦人。这在 Flash 上从未发生过，为什么它在 HTML5 上发生？让我们谈谈 0 伤害，为什么它会发生，为什么我们之前没有这个错误，以及为什么我们现在有。它与游戏的 HTML5 版本无关。它发生是因为我们发布的皮肤以及反作弊系统工作方式的许多细微差别。在大多数情况下，0 伤害错误发生在反作弊系统认为你的行为可疑时。如果你知道我们游戏中的反作弊系统是如何工作的，你就知道它不是客户端的，而是服务器端的。游戏客户端告诉服务器你的坦克做了什么，服务器检查这些动作是否正常，并告诉其他坦克你的坦克做了什么。解释它是如何工作的以及为什么会导致这个错误会很困难，但我试一试。所以我们有通信协议，当你射击时，它会将你的坦克位置发送到服务器进行检查。最近的一些更新影响了位置的准确性，你知道，我们的弹丸飞行时间比平时长，而且由于像危机这样的新无人机，坦克移动得更快。这一切都影响了位置的准确性。在不久的将来，我们将修复它。当你观看此视频时，它很可能已经修复了。所以我们将提高位置准确性，但这还不是全部。你的坦克坐标到底发生了什么？在你的客户端中，你的坦克移动平稳，而对于服务器来说，它离散移动——一阵一阵的。因此，有时，当你的客户端位置和服务器位置不同时，反作弊可能会发现它可疑。你射击地点的坐标和你射击地点的坐标被发送到服务器。服务器计算你的坦克可能处于的一系列位置，以便能够射击到那个地方，并检查你的位置（根据服务器）是否在这个范围内。如果不是，服务器会将其视为可疑活动，你造成 0 伤害。此外，新皮肤存在问题，无论它们是否是高清的。它们有一个非常具体的问题，与弹丸出现的位置有关。我们已经移动了这个点，以便新皮肤看起来更好。然而，我们发现服务器忽略了这些新皮肤，它从默认皮肤中获取该点的坐标。此外，一些炮塔的默认皮肤也已经更改，而其他炮塔尚未更改。这足以让客户端和服务器中的坐标不同。它经常发生在激光炮炮塔上，其中一个激光炮皮肤使你的炮塔更长，所以你可以想象在这种情况下坐标会有多大不同。这就是这两个问题的原因：0 伤害和激光炮的穿透。我们将为所有炮塔修复它。这还不是全部。当弹丸向前移动 1 帧时，它会检查该位置是否有任何物体，如果有，它会告诉服务器它击中了什么。但有时这在服务器看来就像弹丸改变位置太快，从一个点传送到另一个点。想象一下你在近距离射击。弹丸检查到距离你 1 帧的地方有一个物体。它告诉服务器它击中了什么。所以反作弊认为这个弹丸移动了 0 米的距离，它将伤害除以 0，结果得到 0 伤害。类似的事情发生在轰天炮身上，取决于你射击敌方坦克的哪个部位，客户端坐标和服务器坐标之间的差异太大。我们还有马格南……它发送的不是目标的坐标，而是弹丸轨迹点的坐标，因此反作弊系统更难检查这次射击是否可疑。你可能会问，为什么它发生在马格南身上？因为这个问题在马格南发布之前就开始出现了。所以我们在发布马格南时故意破坏了它，我们在发布马格南时故意破坏了它，以便当我们修复这个问题时，马格南能以正确的方式工作，但出了点问题，在修复这个错误时，马格南仍然无法正常工作。无论谁在读它，抱歉，即使我不明白他在说什么，所以我尽量把它翻译得尽可能易懂。此外，我们还有带有火箭的火箭炮。它的火箭有这么大。如果你在近距离射击对手，TL:DR；我们很快就会修复它。
