HTML5 客户端安装指南

Windows 操作系统

Windows 用户指南；

剧透

1. 下载客户端的安装包/可执行文件

2. 在你的电脑/浏览器的“下载”文件夹中找到、打开并运行它


3. 按照（如有提示）屏幕上的软件安装向导完成操作并安装客户端


4. HTML5 客户端现已成功安装在你的电脑上。只需从“开始”菜单、桌面启动它，或在搜索栏中搜索它，即可开始享受游戏。


Mac 操作系统

Mac 用户指南；

剧透

1. 下载客户端安装包

2. 点击文件并让其验证

3. 将 3D 坦克图标拖动并移动到“应用程序”文件夹中

4. 重要提示：一旦你成功将 3D 坦克的 HTML5 客户端拖入“应用程序”文件夹，如果你的电脑上还有其他版本的游戏客户端，弹出窗口将提示你采取行动。如果你已经有了游戏的 Flash 客户端版本，请选择“保留两者”，以允许在你的电脑上同时保留 Flash 和 HTML5 版本。如果这是你拥有的唯一版本，请进入你的应用程序文件夹并直接打开该应用程序。

5. 你必须进入防火墙设置并允许 3D 坦克运行。要进入防火墙，请按照以下步骤操作；

  • 打开“系统偏好设置”
  • 打开“安全与隐私”并转到“通用”
  • 找到 3D 坦克并点击“仍然打开”

6. HTML5 客户端现已成功安装。尽情享受吧！
