底盘

  你知道吗...

该游戏共有六个地图，在控制点模式下，每个地图都有相同数量的点 - 空中楼阁、Deck 9、十字角斗场、围攻、遗弃工厂和黑色城堡。
有四个地图 - 峡谷、城镇角斗场、延迟 Pong 和山岗 - 完全不包含控制点模式。
此外，城镇角斗场是唯一一个在互夺军旗和控制点模式中不可游玩的地图。

底盘是你在"3D坦克"中的坦克的一部分，与炮塔、无人机、迷彩和防御模块一起。它们是玩家可能选择的游戏风格的少校竞争者。它们在速度、力量和重量上有所不同。

在战斗中，你的坦克可以通过这些默认控制进行操作。←→↑↓或W一个SD. 使用移位激活您的底盘的过速装置。这些按键绑定可以在设置菜单。

上将信息

底盘可以从中获得幸运礼盒和抢先体验提供在商店例外是特殊底盘«剑圣»: 它是由游戏在某些战斗模式中自动授予的，因此不可能将其放入你的车库。

* - 一些特殊账号可能拥有剑圣。

每个底盘都有其独特的能力 -过速装置. 例如，一个黄蜂轻甲可以种植一个毁灭性的炸弹，能够眩晕玩家，以及一个十字军可以释放出一种战术冰柱，能够撕裂护甲。这些能力对游戏玩法有显著影响，因此在战斗中应该明智地使用它们。

外观可以通过装备进行修改皮肤.装备改造在车库中的组件装备会影响底盘参数。

底盘升级

每个底盘有7个等级的改装。升级从等级1（Mk1）开始，这是最弱的改装。所有底盘都可以被改装。升级到 Mk7 用于水晶。但是你不能一次性购买所有东西：要解锁新的修改，你需要达到所需的等级下一个修改可用。

每个修改可以逐步升级，直到下一个可用。最高修改在Mk7-20处刻有«MAX»的字样。

您可以在文章中看到底盘在旧图形中的样子（2012年9月6日），以及在再平衡之前（2012年12月17日）它们具备的特征。底盘再平衡前.

轻型底盘

另见:Light 底盘：如何选择，如何使用

黄蜂轻甲

轻型底盘。它具有惊人的灵活性和表面抓地力。

过速装置植入了一个紧凑型的热核炸弹，带有定时器。除了造成良好的伤害，这样的炸弹还会带来大量不幸的后果。敌人将被迫做出反应并寻找掩护。

对于初学者来说，小型轻甲底盘«黄蜂轻甲»可能看起来微不足道。小的尺寸和高速在战场上带来了真正的自由。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速度且非常灵活。过速装置可以同时摧毁半径内的所有红队。
缺点
差的护甲和质量。当使用后坐力高的炮塔时，很容易翻车。


霍珀

光悬停底盘。它非常快速，但在复杂表面上移动时并不太稳定。

过速装置发射位于底盘下方的引爆跳跃引擎。它使敌方坦克眩晕，并将«霍珀»抛入空中，允许跳过障碍。

«霍珀»悬浮坦克使用最高可用的反铁塔发动机资源，让坦克手追求他们的梦想——飞向远离危险的天空。你是否敢于突破敌人的防线？«霍珀»可以帮助你。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速度，可以横向移动并在水平职位上稳定。多亏了过速装置，可以飞得很高并在关键时刻抓住金箱子。
缺点
糟糕的护甲和质量。由于侧向闪避，机械师很容易被推开。目前在跳跃时很容易被远程摧毁。

蜂王

轻量化底盘。高度紧凑。转弯时可以短暂漂移。

过速装置允许坦克手通过热成像瞄准器探测到任何障碍物后面的敌军坦克，并使用多通道通信系统向蓝队发送命令。过速装置的激活还会为您的坦克施加«增压»状态效果，使其对对手造成暴击伤害。

«蜂王»是一款轻巧的底盘，具有流线型曲线。它移动得优雅而顺畅，能在障碍物之间小心地滑动。侦察作战所需的顶级动态属性可实现闪电般的快速攻击。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速度且非常灵活。过速装置让整个团队能够看到障碍物后的红队，并削弱所有红队的防护。
缺点
差的护甲和质量。当使用后坐力高的炮塔时，很容易翻车。

中底盘

另见:中底盘：如何选择，如何使用

维京

中间底盘被跟踪。对于一个中等底盘来说，它的质量相当小，且简介较低。

它的《狂暴反应堆》过速装置显著提高了炮塔的伤害，这要归功于更高的射速。在敌方团队意识到对手有多危险之前，你可以从侧翼攻击，摧毁多个目标。如果你正面攻击敌人，你可能会浪费过速装置的充能。

这个底盘融合了所有最新的军事技术，结合成一件工程艺术的作品。增强的护甲和第二代履带系统使这个底盘成为一款出色的通用坦克。这个底盘可以用于任何类型的战斗任务：从战略防御到全方位突袭。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
多功能底盘具有中等速度和护甲。过速装置可以在短时间内快速消灭大量红队。
缺点
小底盘高度。质量等于轻底盘。过速装置不适合在近射程内造成伤害的炮塔。


十字军

中空底盘。它的特点是良好的通过性和紧凑的轮廓。

它的「Cryo 爆破手」过速装置通过冷却水分几乎从无到有地制造出炮弹，然后射出一个巨大的穿甲冰锥。在被这块冰击中后，对手的坦克会冰冻，令其易于被击毁，如果不能完全摧毁的话，也能轻松结束坦克的行动。

«十字军»底盘属于第二代悬浮坦克。为了创造一个紧凑但强大的底盘，并具备攻击能力，采用了许多巧妙的解决方案。第二代反重力的紧凑性使得工程师能够为坦克装备足够的护甲，使其成为一个中型底盘，同时不降低其速度。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
可以横向移动并在水平职位上稳定。过速装置可以快速攻击多个远程红队。
缺点
如果红队靠得太近，有可能未察觉就自毁于过速装置。由于横向移动，机械师很容易被推开。

猎人中甲

中 tracked 底盘。它在所有中底盘中具有最高的机动性。

它的过速装置产生电磁脉冲，损坏并且更重要的是，能在短时间内使敌方坦克失去行动能力。没有特殊防护，敌人完全无力抵抗，最终被消灭。

这个底盘，覆盖着加固的装甲钢，是最通用的。它将完美适合喜欢积极、动态战斗的玩家，也适合那些喜欢位置战的坦克手。无论是保护一个据点，还是在一场城市废墟中的激烈战斗中，你都不会无所事事。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
多功能底盘具有中等速度和护甲。实用的过速装置可以造成伤害并帮助团队。
缺点
快速移动时可能会失控。

圣骑士

中悬浮底盘。它在所有悬浮底盘中具有最大的机动性。

其过速装置效果应用在底盘周围的射程内，使友方坦克的护甲极化。附近的蓝队获得对某些状态效果的免疫，并持续获得治疗，而负面状态效果则施加在敌方坦克上。

«圣骑士»悬浮坦克是作为支援坦克而设计的，以帮助突袭小组，保持小组的速度至关重要。与重型坦克不同，圣骑士使用中等护甲和第二代反重力引擎，能够跟上轻型坦克的速度。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
可以横向移动并在水平职位上稳定。过速装置可以保护免受负面状态影响并为团队提供治疗。
缺点
过速装置充电非常慢。由于侧向移动，机械师容易被推开。

独裁者

中 tracked 底盘。它对于一辆中坦克来说非常沉重和坚固，并且因其增加的尺寸而闻名。

它的过速装置激活所有道具对自己和所有附近的蓝队的效果，施加«增压»状态效果，并在红队周围制造冰风暴。

由于采用了改进的主动防护系统和最新的复合材料，这款底盘成功地吸收了任何口径弹药爆炸的能量，同时仍然保持为中底盘。这样的底盘的拥有者可以轻松带领团队进行全面突袭，抵御攻击的冲击。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
大质量和卓越稳定性结合了高速。大的底盘高度允许在某些情况下上射击岗位。过速装置为整个团队提供额外道具，在效果半径内。
缺点
大的高度使它成为一个易于攻击的目标，而在近距离射程内，有时难以击中敌人。在大多数情况下，攻击敌方基地时它并不太实用。

重型底盘

另见:重型底盘：如何选择，如何使用

泰坦

重型履带底盘。其紧凑的尺寸和改进的加速动态使其与其他重型坦克不同。

«泰坦» 底盘在过速装置激活时放置一个保护圆顶发电机。这个圆顶显著减少所有盟友坦克受到的伤害，并逐渐修复它们。对手可以通过使用其他过速装置等特殊攻击来攻击发电机，从而停用圆顶。

用于最严酷防御任务的重型底盘。由最强的钛合金与完美的结构和几何形状结合而成，使«泰坦»成为能够抵挡甚至重口径炮塔的完美盾牌。装备这个底盘的坦克手们可以直接驶入地狱。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
优秀的护甲和高质量允许轻松推开红队并保护盟友坦克。过速装置为整个团队在效果半径内提供保护。
缺点
非常低的速度使得躲避攻击变得不可能。防护罩可以与其他过速装置一起关闭。

阿瑞斯

重型悬浮底盘。在遭受攻击时仍然保持稳定。

最强大的《BFG大炮》直接安装在《阿瑞斯》底盘上。这个过速装置发射一个特殊的等离子球，攻击敌方坦克并治疗蓝队。

悬浮坦克«阿瑞斯»成为首个具有反铁塔功能的底盘。其令人印象深刻的重量和第一代反重力机的比例使得«阿瑞斯»进入了重型底盘类别。在战斗中想要翻转这辆悬浮坦克，你需要付出非常大的努力。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
很棒的护甲和高质量。可以侧向移动，并在水平位置稳定。过速装置可以在一条直线上摧毁多个红队并治愈蓝队。
缺点
坦克的体积庞大，占据了大量的宇宙，慢且明显。如果过速装置在离坦克太近的地方爆炸，就会导致自毁。

猛犸象

重型履带底盘。它拥有最多的装甲、质量和稳定性。

«猛犸象»底盘的过速装置提高了它的速度，瞬间修复它，并赋予它一种致命的力量，可以粉碎敌方坦克并在途中特别摧毁地雷。

一座在轨道上移动的要塞。重型底盘中的旗舰。坚固的重型钢铁，混凝土护甲加固，速度缓慢，但能够承受数个敌方炮塔的火力。它不可阻挡——即使你用坦克封锁道路，坦克也会在这个怪物的履带下燃烧。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
良好的健康和高质量可以轻松阻止红队，同时保持稳定。过速装置将坦克在短时间内变成绝对的毁灭机器。
缺点
非常低的速度和机动性，再加上底盘的大尺寸，使得它难以躲避射击。

特殊底盘

剑圣

他会活下来的！

超重型底盘。它具有特殊的绝缘材料，以便在危险区域工作。

剑圣的过速装置释放一个能量场，将附近的红队全部击退，并恢复底盘的健康。由于使用了大量材料，其他修复坦克的方法非常缓慢。

重型实验机。入场严格限制。

  • 防护
  • 顶级速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
非常好的健康，质量和速度。能够长时间承受持续的敌人火力。
缺点
非常大的坦克，因此它可能无法通过某些通道。在高速下很容易失去对坦克的控制。
