底盘是《3D坦克》中你坦克的一部分，与炮塔、无人机、迷彩和防御模块一起。它们是玩家选择游戏风格的重要因素。底盘在速度、强度和重量上有所不同。

在战斗中，你的坦克可以通过以下默认控制来操作：←→↑↓或者WASD. 使用Shift要激活你的底盘的过速装置。这些快捷键可以在设置菜单。

一般信息

底盘可以从…获得幸运礼盒和抢先体验的优惠商店例外是特殊底盘«剑圣»: 它在某些战斗模式中由游戏自动授予，因此不可能将其放入您的车库。

* - 一些特殊账号可能拥有剑圣。

每个底盘都有其独特的能力 -过速装置. 例如，一个黄蜂轻甲可以放置一个能使玩家眩晕的毁灭性炸弹，以及一个十字军可以释放一个战术战斗冰柱，能撕裂护甲。这些技能会显著影响游戏玩法，因此在战斗中应明智使用。

外观可以通过装备来修改皮肤.装备改造是车库中影响底盘参数的组件装备。

底盘升级

每个底盘有7个等级的改装。升级从等级1 (Mk1)开始，这是最弱的改装。所有底盘都可以升级的去 Mk7 购买水晶。但是你无法一次性购买所有东西：要解锁新的改装，你需要达到所需条件等级以便下一次修改可用。

每次改装都可以逐步升级，直到下一次改装可用。最高级别的改装在 Mk7-20 上刻有“MAX”字样。

您可以在文章中看到底盘在旧图形（2012年9月6日）中的样子，以及它们在重新平衡前（2012年12月17日）具有的特征底盘在重新平衡之前.

Light 底盘

黄蜂轻甲

轻型履带底盘。它在地面上的机动性和抓地力都非常出色。

过速装置安装了一个带有定时器的紧凑型热核炸弹。除了造成巨大的伤害外，这种炸弹还会遗留大量不幸的后果。敌人将被迫做出反应并寻找掩护。

小型轻甲「黄蜂轻甲」底盘乍看之下可能只会被初学者轻视。小巧的尺寸和高速性能赋予了战场上的真正自由。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速且非常机动。过速装置可以在爆炸半径内同时摧毁所有红队成员。
缺点
护甲和质量差。使用高后坐力的炮塔时很容易翻车。


霍珀

轻型悬浮底盘。它非常快，但在复杂表面上移动时不太稳定。

过速装置启动安装在底盘下方的爆炸跳跃引擎。它使敌方坦克眩晕，并将«霍珀»抛向空中，使其能够跳过障碍物。

«霍珀»悬浮坦克使用最高级的反铁塔引擎资源，让坦克手追逐他们的梦想——飞向天空，远离危险。你敢突破敌人的防线吗？«霍珀»可以帮助你。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速，可以侧向平移并在水平位置稳定。多亏了过速装置，你可以飞得很高，并在关键时刻抓住金箱子。
缺点
护甲和质量很差。由于侧向移动机制，很容易被推开。跳跃时容易从远处被摧毁。

蜂王

轻型履带底盘。高度紧凑。转弯时可以短暂漂移。

过速装置允许坦克手使用热能瞄准镜探测所有被障碍物遮挡的敌方坦克，并通过多通道通信系统向蓝队发送命令。激活过速装置还会为你的坦克施加“增压”状态效果，使其能够对敌人造成暴击伤害。

«蜂王» 是一款轻型底盘，拥有流线型的空气动力学曲线。它移动优雅而顺畅，能在障碍物之间小心地滑行。侦察作战所需的顶级动态性能，使其能够发动闪电般迅速的攻击。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
高速且非常灵活。过速装置让整个团队能够看到障碍物后的红队，并削弱所有红队的防护。
缺点
护甲和质量差。使用高后坐力的炮塔时很容易翻车。

中型底盘

另请参见:Medium 底盘：如何选择，如何使用

维京

中型履带底盘。对于一个中型底盘来说，它的质量相当小，并且简介低。

其«狂暴反应堆»过速装置由于射速提高，显著增加了炮塔的伤害。你可以通过从侧面攻击敌方团队，在他们意识到对手有多危险之前摧毁多个目标。如果正面攻击敌人，你可能会浪费过速装置的能量。

这款底盘融合了所有最新的军事技术，是一件集工程艺术于一体的作品。增强的护甲和第二代履带使这款底盘成为一款出色的通用坦克。该底盘可用于任何类型的战斗任务：从战略防御到全线突袭。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
多功能底盘，具有中速和护甲。过速装置可以在短时间内快速摧毁大量红队成员。
缺点
底盘高度小。质量等于轻型底盘。过速装置不适合在近距离射程造成伤害的炮塔。


十字军

中型悬浮底盘。其特点是通过性良好且外形紧凑。

它的«Cryo 爆破手»过速装置通过冷却水分几乎从无中生成一枚弹丸，然后射出一根巨大的穿甲冰柱。被这块冰击中后，敌方的坦克会遭遇冰风暴，使其容易被击败，如果不是直接摧毁的话。

“十字军”底盘属于第二代悬浮坦克。为了打造一款紧凑却强大的底盘并具备攻击能力，使用了许多巧妙的解决方案。第二代反重力装置的紧凑性使工程师能够为坦克配备足够的护甲，使其成为中型底盘而不降低速度。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
可以横向平移并在水平位置稳定。过速装置让你能够快速远程攻击多个红队。
缺点
如果红队靠得太近，有可能没有注意到他们的接近而被过速装置自毁。侧向平移机制很容易被利用。

猎人中甲

中型履带底盘。在所有中型底盘中，它的机动性最强。

其过速装置会产生电磁脉冲，造成损伤，更重要的是，可以在短时间内使敌方坦克失去行动能力。如果没有特殊防护，敌方将完全无力反击，任人宰割。

这款底盘覆盖有加强型装甲钢，是最通用的。它非常适合喜欢积极、动态战斗的玩家，也适合喜欢策略性玩法的坦克手们。无论是保护据点，还是为废墟中的城市进行激烈战斗，你都不会无事可做。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
多功能底盘，具有中等速度和护甲。实用的过速装置，可以造成伤害并帮助团队。
缺点
高速行驶时可能会打滑。

圣骑士

中型悬浮底盘。它在所有悬浮底盘中机动性最强。

其过速装置效果会作用在底盘周围的射程范围内，使友方坦克的护甲极化。附近的蓝队单位可以免疫某些状态效果并持续恢复生命，同时敌方坦克会受到负面状态效果影响。

«圣骑士»悬浮坦克被设计为支援坦克，以协助突袭小组，而保持小组的速度至关重要。与重型坦克不同，圣骑士拥有中等护甲，并使用第二代反重力引擎，使其能够跟上甚至轻型坦克的速度。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
可以横向平移并在水平方向上稳定。过速装置可以防止负面状态效果并治疗团队。
缺点
过速装置充能非常缓慢。由于横向平移机制，它很容易被推动。

独裁者

中型履带底盘。对于中型坦克来说，它非常重且坚固，同时也因其较大的尺寸而闻名。

它的过速装置会激活自身及所有附近蓝队成员的道具效果，施加«增压»状态效果，并对周围的红队造成冰风暴效果。

得益于改进的主动防护系统以及在制造中使用的最新复合材料，这款底盘能够成功吸收各种口径炮弹的爆炸能量，同时仍保持为中型底盘。拥有这样的底盘的车主可以轻松带领团队进行全面突袭，承受住攻击的主要冲击。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
巨大的质量和出色的稳定性结合了高速。大的底盘高度允许在某些情况下进入射击姿势。过速装置在效果半径内为整个团队提供额外道具。
缺点
较高的高度使其成为一个容易的目标，在近距离射程内，有时很容易错过对敌人的攻击。在大多数情况下，攻击敌方基地时它并不太实用。

重型底盘

另请参见:重型底盘：如何选择，如何使用

泰坦

重型履带底盘。其紧凑的尺寸和改进的加速动力学使其有别于其他重型坦克。

“泰坦”底盘在过速装置激活时会放置一个保护罩发生器。该保护罩可以显著降低所有友方坦克受到的伤害，并逐渐修复它们。对手可以通过使用特殊攻击，例如其他过速装置，攻击发生器来关闭保护罩。

坚固的底盘用于最艰难的防御任务。由最坚固的钛合金制成的护甲，结合完美的结构和几何设计，使“泰坦”成为即使是大口径炮塔也无法击穿的完美盾牌。装备这款底盘的坦克手可以直接冲入火海。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
优秀的护甲和高质量允许轻松推动红队并保护友方坦克。过速装置在效果半径内为整个团队提供防护。
缺点
非常低的速度，这使得躲避射击几乎不可能。防护穹顶可以通过其他过速装置关闭。

阿瑞斯

重型悬浮底盘。即使在火力下行驶，它仍然保持稳定。

最强大的 «BFG Cannon» 直接内置于 «阿瑞斯» 底盘。这种过速装置发射一个特殊的等离子球，对敌方坦克造成雷电攻击，并为蓝队提供治疗。

悬浮坦克«阿瑞斯»成为第一个拥有反塔功能的底盘。其令人印象深刻的重量和第一代反重力装置的比例，使«阿瑞斯»进入了重型底盘类别。要在战斗中翻转这辆悬浮坦克，你需要非常努力。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
很棒的护甲和高质量。可以横向移动并在水平位置稳定。过速装置可以摧毁多名红队成员并直线治疗蓝队。
缺点
占用大量空间的大型坦克行动缓慢且显眼。如果超速装置在坦克附近爆炸，会导致自毁。

猛犸象

重型履带底盘。它拥有最多的护甲、质量和稳定性。

«猛犸象» 底盘的过速装置可以提升其速度，立即修复它，并赋予其致命的力量，碾压敌方坦克并沿途摧毁地雷。

一座在轨道上移动的堡垒。在重型底盘中是旗舰。坚固的重量块，由混凝土护甲加固，速度缓慢，但能够在同时承受几座敌方炮塔的火力下存活。它是不可阻挡的——即使你用坦克封锁道路，它们也会在这怪物的履带下燃烧殆尽。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
出色的生命值和高质量护甲允许轻松阻挡红队而保持稳定。过速装置在短时间内将坦克变成绝对的破坏机器。
缺点
非常低的速度和机动性，再加上底盘的巨大尺寸，这使得很难躲避射击。

特殊底盘

剑圣

他会活下去！

超重型底盘。它具有特殊绝缘，可在危险区域使用。

剑圣的过速装置会释放出一个能量场，将所有附近的红队击飞，并恢复底盘的生命值。由于使用了大量材料，其他修复坦克的方法非常缓慢。

重型实验机器。入内严格限制。

  • 防护
  • 最高速度
  • 转向灵活度
  • 重量
  • 启动加速度
优点
非常好的健康、质量和速度。能够长时间应对连续的敌方火力。
缺点
非常大的坦克，因此可能无法通过某些通道。高速时很容易失去对坦克的控制。
