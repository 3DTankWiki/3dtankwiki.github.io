折扣特价优惠是指以折扣价购买商品的优惠。在车库打折产品可以通过标签轻松识别位于左上角，清晰显示折扣百分比。最终折扣价格会突出显示在“购买”按钮上。查看时炮塔,底盘，或者模块预览时，与 Mark 等级对应的价格保持原值。这允许玩家清晰地看到正在应用的折扣。折扣来源多样，通常仅在有限时间内有效。物品显示的价格始终反映所有当前有效的折扣。

当单个商品同时有效多个折扣时，它们不会简单地加起来。相反，它们充当商品原价的乘数。这种方法可以防止折扣超过 100% 的可能性。由于乘法顺序不影响结果，因此应用折扣的顺序无关紧要。计算综合折扣的公式以及说明性示例在”计算折扣值得注意的是，同时出现多重折扣的情况相对罕见。

注意
销售、折扣和庆祝活动的具体日期和百分比通常不会提前很久公布。通常，它们只会在开始的前几天才公开，有时甚至会在没有任何提前通知的情况下开始。为了了解最新动态，玩家应定期查看游戏内的信息。通信面板用于新闻和公告。

折扣类型

节日折扣

这些折扣是为了纪念重要日期而在有限时间内推出的，例如元旦、3D坦克的周年纪念、坦克手节以及其他特殊场合。折扣通常为30%或50%，尽管这些数字基于以往的活动，并可能根据开发者的判断而有所变动。此外，还会有其他折扣活动。车库商品、节日促销还可能包括价格折扣迷彩,道具,通行证,礼物，和/或微升级，由开发者决定。微型升级的折扣可以应用于步骤成本或立即完成，也可以单独应用于两者。

不再经常提供的折扣

主题折扣

注意
它们在游戏中出现得非常少，这种类型的折扣可能无法在未来世界中再现。

通常不与节假日绑定，它们可能会在任何时间出现，这取决于社区经理和游戏营销人员的计划。主题折扣可以适用于一小部分物品，可能是不同类别的，并且持续时间有限。例如：2013年11月29日至30日对地雷的90%折扣，每小时70%的折扣维京2014年10月26日，迷彩的折扣恋爱中在情人节等节日，主题折扣相当珍奇。

节日和主题折扣的出现完全取决于开发者。这些折扣活动会在游戏官网主页、论坛主题、进入游戏时的提醒，以及社交网络和其他官方渠道的信息中公布。不要上当相信聊天中玩家发来的关于“前所未有的折扣大小”的钓鱼信息！

计算折扣

如果某件商品同时适用多种折扣，那么要计算该商品的最终价值，需要使用以下公式：

  1. 确定每个折扣下物品的残值，以原价的百分比表示。这可以通过用100%减去每个折扣百分比来完成。例如，20%的折扣会导致物品的残值为原价的80%，而35%的折扣则会得到65%的残值。
  2. 将这些百分比转换为小数分数，通过将它们除以100。使用前面的例子，80%变为0.8，65%变为0.65。
  3. 将这些小数分数相乘，然后将结果乘以物品的初始价值。例如，如果磁力炮 Mk7 的初始价格是 150,000 水晶，计算方法为：150,000 x 0.8 x 0.65 = 78,000。将任何小数结果四舍五入到最接近的整数。
  4. 因此，考虑到两种折扣后，磁力炮 Mk7 的折后价格为 78,000 水晶。整体折扣计算为 (1 - (0.8 x 0.65)) x 100% = 48%.
