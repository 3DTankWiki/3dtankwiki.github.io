折扣是购买降价商品的特价优惠。在车库打折商品可以通过标签轻松识别位于左上角，清晰地显示折扣百分比。最终折扣后的价格明显显示在“购买”按钮上。当查看时炮塔,底盘，或者模块预览时，与 Mark 等级对应的价格保持原价。这允许玩家轻松看到正在应用的折扣。折扣来自各种来源，通常在有限时间内有效。商品显示的价格始终反映所有当前有效的折扣。

当单个商品同时存在多个折扣时，它们不会简单地相加。相反，它们会作为商品原价的倍数来运算。这种方法可以防止折扣总额超过100%。由于乘法的顺序不影响结果，因此折扣的应用顺序无关紧要。计算组合折扣的公式及示例详见“计算折扣值得注意的是，同时出现多重折扣的情况相对罕见。

注意
具体的销售、折扣和庆祝活动的日期和百分比通常不会提前很久公布。通常，它们只会在开始前几天才被透露，有时甚至在没有任何事先通知的情况下就开始。为了随时了解最新动态，玩家应定期查看游戏内的消息。通信面板用于新闻和公告。

折扣类型

节日折扣

这些折扣是为纪念重要日期而在有限时间内推出的，例如元旦、3D坦克的周年纪念、坦克手日以及其他特殊场合。折扣百分比通常为30%或50%，尽管这些数字是基于过去的活动，并且可能会根据开发者的决定而变化。与此同时，折扣也适用于车库商品，节日促销还可能包括价格降低迷彩,道具,通行证,礼物，和/或微升级由开发者决定。微升级的折扣可以应用于步骤成本或立即完成，或单独应用于两者。

不再频繁的折扣

主题折扣

注意
它们在游戏中非常罕见，而且这种类型的折扣可能在未来世界中不会再出现。

通常不与节假日挂钩，它们可能会在任何时间出现，具体取决于社区管理员和游戏营销人员的计划。主题折扣可以适用于一小部分商品，可能是多样化的，并且持续时间有限。例子：2013年11月29日至30日，地雷九折优惠，每小时70%的折扣用于维京2014年10月26日，迷彩享受折扣恋爱中在情人节等日子，主题折扣非常珍稀。

节日和主题折扣的出现完全取决于开发者。这些折扣活动会在游戏主页发布公告，在论坛开设专题，在进入游戏时发送提醒，并通过社交网络及其他官方渠道发布信息。不要轻信聊天中玩家发送的关于“前所未有折扣大小”的钓鱼信息！

计算折扣

如果多种折扣同时适用于某件商品，那么要计算该商品的最终价值，需要使用以下公式：

  1. 确定每种折扣下物品的剩余价值，以原价的百分比表示。这可以通过用100%减去每个折扣百分比来实现。例如，20%的折扣会使剩余价值为原价的80%，而35%的折扣会使剩余价值为65%。
  2. 将这些百分比转换为小数分数，通过将它们除以100。使用前面的例子，80%变为0.8，65%变为0.65。
  3. 将这些小数分数相乘，然后将结果乘以物品的初始价值。例如，如果一台磁力炮 Mk7 的初始价格为 150,000 水晶，计算方法为：150,000 x 0.8 x 0.65 = 78,000。将任何小数结果四舍五入到最接近的整数。
  4. 因此，考虑到两种折扣后，磁力炮 Mk7 的折后价格为 78,000 水晶。总体折扣计算为 (1 - (0.8 x 0.65)) x 100% = 48%.
