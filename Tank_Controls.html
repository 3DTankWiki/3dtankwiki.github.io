坦克操作

另请参阅：如何开始


游戏中有两种控制坦克的方法：键盘和键盘+鼠标。仅使用键盘，你可以用箭头键移动坦克，用空格键射击，并用 Z 和 X 键转动炮塔。

使用鼠标时，它可以控制炮塔和摄像机。射击可以用鼠标左键完成。

HTML5 控制键

注意！下方的键盘设置是标准设置。你可以在游戏设置中将控制重置为不同的按键。


控制 操作
←→↑↓ 或 WASD 坦克移动和转向（当 NUMLOCK 关闭时，小键盘上的箭头键也有效）
Z 和 X 或, 和 . 向左/向右转动炮塔或升降马格南炮管
C 或 / 炮塔居中或将马格南炮管角度设置为标准
SPACE 射击
DELETE 或 CTRL + \ 自毁。会有延迟
从 1 到 6 的按键 使用道具（按键顺序：维修工具、护甲提升、伤害提升、速度提升、地雷、金箱子）
左 SHIFT 使用过速装置和核能道具
F 投掷手榴弹 - 按住按钮可在投掷前查看手榴弹的轨迹。
R 或 V 丢弃旗子或球。当玩家持有旗子或球时，投掷手榴弹被禁用。（功能适用于互夺军旗、橄榄球和突袭战斗模式）
TAB 1) 查看当前战斗统计。表格中包含战斗中的所有玩家及其战斗指标。 2) 以表格形式显示战斗中玩家的信息：装备分数、他们使用的防护模块和装备改造，以及他们自己的模块防御哪些炮塔。

3) 当聊天打开时，在团队和常规聊天之间切换（适用于团队战斗模式）。 4) 在大厅打开常规聊天

Q 和 E 或 PAGEUP 和 PAGEDOWN，或[和] 升降摄像机
ENTER 1) 打开聊天发送消息，第二次按 Enter - 发送消息。 2) 确认操作（在对话窗口中）
P 在战斗中暂停。当暂停计时器结束时，你将因不活动而被踢出战斗。
I 在战斗中打开车库
ESC 1) 在战斗中打开菜单。 2) 退出全屏模式、战斗或游戏。 3) 取消操作（在对话窗口中）
NUM+和 NUM-或+和= 增大或减小游戏窗口大小。窗口越小，处理器和显卡的负载就越小。

鼠标控制

鼠标控制可以在设置中的相应选项中开启。

控制 操作
鼠标左键 射击
鼠标向左或向右移动 水平移动摄像机（炮塔将跟随摄像机转动）
滚轮向上和向下 升降摄像机
鼠标右键（按住） 阻止炮塔跟随摄像机转动


在 Google Chrome 中激活

在 Google Chrome 中开启鼠标控制的说明

Google Chrome 是玩 3D 坦克的可能客户端之一。浏览器的安全规则要求获得捕获鼠标的许可，才能使用鼠标控制玩 3D 坦克。

在 Google Chrome 中首次进入战斗时，浏览器会询问是否锁定并使用鼠标：

点击“每次访问时允许”。这只需要一次；浏览器会记住你的选择。

如果窗口没有出现且鼠标控制不起作用，或者你选择了“从不允许”，该怎么办：

  1. 确保在“控制”部分的设置中选择了“鼠标 + 键盘”
  2. 在游戏中，点击地址栏左侧的设置图标
  3. 激活鼠标控制
  4. 重新加载游戏页面

开启鼠标控制的替代方法：

  1. 在游戏选项卡上时，按地址栏左侧的设置图标
  2. 选择“网站设置”
  3. 在设置列表中，找到“鼠标锁定和使用”
  4. 选择“允许”

如果坦克翻转或侧翻怎么办？

  1. 如果你的坦克侧翻且拥有带后坐力的炮塔，你可以尝试射击地面。
  2. 如果这没有帮助，尝试请求团队中的玩家帮助将你的坦克翻转到正常位置。
  3. 如果什么都没用，按 DELETE 键自毁（你的击杀面板中的失败次数总数将增加 1）。
    • 如果不起作用，尝试 CTRL + \按键组合。
    • 如果你没有 DELETE 键（例如 Mac OS 用户），请按 FN + BACKSPACE 按键组合。
  4. 使用可以在游戏设置中开启/关闭的自动自毁。
  5. 如果其他方法都无效，剩下的唯一办法就是刷新页面。

建议：带后坐力的炮塔可以显著降低从悬崖跳下时坦克翻转的几率。向倾斜的方向射击。

可能的键盘问题

主要文章：键盘问题

应该注意的是，并非所有键盘都能在同时按下多个键时工作，特别是如果“箭头”键在其中。如果坦克在移动时无法同时射击和转动炮塔，就会注意到这一点。这个问题可以通过重置控制并使用 WASD 键的替代键盘设置，或者使用在同时按下多个键时不会出现问题的键盘来解决。
