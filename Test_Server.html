目的

测试服务器的目的是在对测试员舒适的条件下检查新版本并搜索错误。水晶、购买的物品以及在测试服务器上收到的任何其他游戏内物品永远不会添加到你在主服务器上的账号中。

邀请

如果主服务器上没有进行大规模测试，则进入测试服务器仅限于邀请。只有一个人可以使用它——即收到邀请的那个人。

邀请仅提供给测试员和测试员候选人。

邀请仅由开发人员提供。

管理层从不向不属于测试员和测试员候选人小组的玩家提供邀请。如果测试服务器需要更多玩家，则会开始公开测试（大规模测试）。

如果你在某处“注册”并收到了“邀请”——你将无法获得官方测试服务器的访问权限。很可能你的账号已被盗，你会发现它因作弊而被封禁或锁定。在这种情况下，你可能是在诈骗（钓鱼）网站上注册了，泄露密码导致你的账号被黑。请记住，获得邀请不需要你的电子邮件或密码。如果你的账号被盗，请联系技术支持。

大规模测试

有时，为了检查新版本等内容，需要大量玩家同时在线。在这种情况下，测试服务器对所有人开放。测试服务器的开放从不提前宣布。

测试服务器命令

有时，特殊的命令仅在测试服务器上可用，可以更改测试账号的参数。通常，它们仅对开发人员有效。

  • “/addcry n” - 为测试服务器上的当前账号添加 n 个水晶；
  • “/addexp n” - 为测试服务器上的当前账号添加 n 点经验值；
  • “/addpremium n” - 为测试服务器上的当前账号添加 n 分钟的 VIP；



注意！命令在主服务器上不起作用。当试图在测试服务器之外使用它们时，你可能会收到警告或封禁。


沟通文化

测试服务器上的行为和游戏规则与主服务器相同。

如何获得测试服务器访问权限

真正的 3D 坦克测试服务器位于链接 https://test.tankionline.com/


注意！所有其他提供的链接都是为了获取你的账号而制作的诈骗链接！


测试服务器的开放没有时间表。

不要忘记测试服务器的资源是有限的。如果你无法进入测试服务器，请稍后再试。在测试期间，如果需要修复某些问题或需要更新，服务器可能会重新加载或停止。

真正的测试服务器有三种状态：

  • 关闭。没有人有访问权限；
  • 对测试员和测试员候选人开放。需要邀请才能进入；
  • 对所有人开放。要进入，需要创建一个新账号并“激活”主服务器上账号的登录名和密码。

如何成为测试员并获得邀请

要申请成为测试员，请在测试员候选人网站上填写申请表。当你填写并发送表格后，你将被添加到特定的 Discord 服务器，你的请求将在那里得到处理。在这个 Discord 服务器上，你将能够直接与测试员管理员交谈。

常见问题解答

问： 什么是邀请码？
答： 测试服务器邀请码是一个字母数字字符串，用于测试员和开发人员在封闭测试时访问测试服务器。该代码就像进入测试服务器的密码。


问： 测试服务器何时向公众开放？（无需邀请码）
答： 测试服务器仅在需要大量玩家基础来测试新功能或错误修复时才会开放。这些需要大量玩家的时间没有时间表，因此了解测试服务器是否开放的唯一方法是自己检查。

问： 为什么测试服务器上的商店被锁定了？
答： 开发人员故意锁定商店，因为那里没有什么需要测试的，并且不希望他们的测试统计数据受到商店中购买行为的影响。它还可以防止玩家用真钱购买物品，因为这些账号在测试后会被删除。


注意！不要在测试服务器商店中用真钱购买任何东西。在测试服务器中获得的所有物品在测试结束后都会被移除，并且不会添加到你的真实账号中。


问： 我在哪里可以获得邀请码？
答： 邀请码仅提供给候选测试员、测试员和开发人员。获得邀请码的唯一方法是成为测试员。


注意！只有候选测试员、测试员和开发人员才能获得邀请码。其他人无法获得邀请码。提供出售邀请码的玩家是在试图诈骗你！
