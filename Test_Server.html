目的

测试服务器的目的是为了在测试人员舒适的条件下检查新发布的版本并搜索错误。水晶，购买的物品，以及在测试服务器上收到的任何其他游戏内物品将永不被添加到你的账号在主服务器上。

邀请函

如果在主服务器上没有进行质量测试，进入服务器的权限仅限于邀请。只有一个人可以使用它 - 那个收到邀请的人。

邀请仅发给测试员和测试员候选人.

邀请函是仅由开发者提供.

管理层从不向不属于测试者和测试候选小组的玩家发出邀请。如果测试服务器需要更多玩家，将进行公开测试（质量测试）。

如果你在某个地方“注册过”并收到过“邀请”，你将无法访问官方测试服务器。很可能你的账号被盗了，你会看到它因作弊而被禁止或封锁。在这种情况下，你可能是在一个诈骗（钓鱼）网站上注册的，泄露你的密码导致了你的账号被黑客入侵。请记住，获取邀请不需要你的电子邮件或密码。如果你的账号被盗了，请联系技术支持.

质量测试

有时候要检查像新版本这样的东西，需要大量玩家同时参与。在这种情况下，测试服务器对所有人开放。测试服务器的开放从不提前通知。

测试服务器的命令

有时特殊命令是仅仅可用于测试服务器，可以更改测试账号的参数。通常这些功能仅针对开发人员。

  • “/addcry n”- 加入n水晶to the current 账号 on the test 服务器;
  • “/addexp n”- 加入n经验值points to the current 账号 on the test 服务器;
  • “/addpremium n”- adds n minutes of贵宾to the current 账号 on the test 服务器;



注意！命令在主服务器上无效。当尝试在测试服务器之外使用它们时，您可能会遇到问题。警告或禁令。

如何成为测试员并获得邀请

要申请成为测试者，请填写申请表格上的测试人员候选人网站。当您填写表单并发送后，您将被添加到一个特殊的Discord服务器将在这里处理您的请求。您将能够在这个 Discord 服务器上直接与测试管理员交谈。

常见问题

问： 什么是邀请代码？
A：测试服务器邀请码是一个由字母数字组成的字符串，供测试者和开发者在进行封闭测试时访问测试服务器。这个代码就像是进入测试服务器的密码。


问： 测试服务器何时会向公众开放？（不需要邀请码）
A： 测试服务器仅在需要大规模玩家基础来测试新功能或修复漏洞时开放。这些需要大量玩家的时间没有固定的时间表，因此唯一知道测试服务器是否开放的方法就是自己检查。

问： 为什么测试服务器上的商店被锁定了？
A： 开发者故意锁定商店，因为那里没有需要测试的内容，并且不需要他们的测试统计数据受到商店中购买行为的影响。这也防止玩家用真实货币购买物品，因为这些账号在测试后会被删除.


注意！请勿在测试服务器商店中使用真钱购买任何物品。在测试结束后，测试服务器中获得的所有物品将被删除，并不会添加到您的真实账号中。


问： 我在哪里可以获得邀请代码？
A： 邀请码仅提供给候选测试员、测试员和开发者获得一个的唯一方式是成为测试员.


注意！只有候选测试员、测试员和开发者可以获得邀请代码。其他人无法获得邀请代码。提供出售邀请代码的玩家正在试图骗你！
