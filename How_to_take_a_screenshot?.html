截图的各种方法

使用 Microsoft Paint

  1. 前往你想要截图的位置。
  2. 按下 PrtScr（Print Screen）键进行全屏截图。或者按 Alt+PrtScr 仅对活动窗口进行截图。
  3. 打开任何图形程序，例如 Paint。（开始 → 程序 → 附件 → 画图。）进入程序后，按 Ctrl+V，或在工具栏中：编辑 –> 粘贴。
  4. 你的截图将出现在画图窗口中。
  5. 点击文件 → 另存为。...
  6. 选择 png 或 jpg 文件格式。
  7. 点击保存。


Windows 截图（替代方法）

在 Windows 8、9、10 和 11 中，你可以通过点击 Win+PrintScrn 进行截图。

此组合键会对当前屏幕进行截图，并默认将其保存为图片文件夹中的 PNG 图像，即图片文件夹（C:\Users\用户名\Pictures\Screenshots）。 每个文件都会自动获得一个名称，例如 'Screenshot (1).png'、'Screenshot (2).png'、'Screenshot (3).png' 等。

在 Mac OS 中截图

你不需要任何额外的软件即可在 Mac OS 中截图。你只需要知道几个（或至少一个）键盘快捷键。你可以在 Mac 的设置中查看所有快捷键（系统偏好设置 → 键盘与鼠标 → 键盘快捷键）。

  1. 要捕获整个桌面，请按 Command-Shift-3。截图将自动保存为桌面上的 PNG 文件。
  2. 要复制整个桌面，请按 Command-Control-Shift-3。截图将被放置在剪贴板上，供你粘贴到另一个程序中。
  3. 要捕获桌面的某一部分，请按 Command-Shift-4。十字准线光标将出现，你可以点击并拖动以选择你想要捕获的区域。当你松开鼠标按钮时，截图将自动保存为桌面上的 PNG 文件。


免费截图程序

互联网上还有许多免费程序可以帮助你截图。

其中一个程序是 Lightshot。

Lightshot 是一款免费的截图软件，适用于 Mac 和 Windows 计算机；提供了一些即时编辑功能，并可以选择将截图保存为文件，或免费上传到他们的网站。

创建一个账户将提供永远不会丢失截图的额外优势，因为 URL 与你的账户绑定，所以你可以在以后的任何时候找到你过去拍摄的任何截图。

  1. 下载并安装 Lightshot。（安装后它将自动运行，并且每次你打开计算机时都会自动启动，因此无需担心打开应用程序的问题。）
  2. 然后，当你想要截图时，只需按下键盘上的 Print Screen 键，并选择你想要截图的区域。你可以通过点击并拖动鼠标来完成此操作。
  3. 然后选择你想要如何分享你的截图。如果你感到困惑，请参考上图。你可以通过点击云按钮在线上传图像 - 然后只需复制并粘贴图像的链接。你可以将图像复制到剪贴板。或者你可以直接将图像保存到你的计算机上。
