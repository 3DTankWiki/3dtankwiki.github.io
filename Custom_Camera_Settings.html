自定义摄像机设置


注意！这些设置不再适用于游戏的新 HTML5 版本。


另请参阅：经验丰富的玩家的建议


自此更新以来，任何玩家都可以自由使用修改后的观察者模式！要使用“无模式观察者”，你需要遵循几个简单的步骤：

步骤 说明
1. 进入一场战斗。
2. 找到你想要的拍照位置。
3. 按下 Ctrl + Shift + K 打开任务选项卡。
4. 输入 vars.set cam_maxmove 0 并按下 Enter。
5. 按下 Ctrl + Shift + K 或 Escape 退出任务选项卡。
6. 从屏幕上开走。

调整垂直摄像机速度

命令

vars.set cam_vspeed 0.5

你可以更改按下 Q 和 E 或 PageUp 和 PageDown 键时视图更改的速度。

为此，进入一场战斗，使用 Ctrl + Shift + K 组合键打开控制台，并输入以下命令：vars.set cam_vspeed 0.5，其中 0.5 是所需的速度。

默认数字是 0.7。增加此值会使摄像机移动得更快，而减小该值会使其移动得更慢。


注意！目前，设置中的任何更改仅在下一次服务器更改前有效，之后它们将重置为默认值，需要再次更改。


调整水平摄像机速度

命令

vars.set cam_maxmove 3

你可以使用控制台更改当你转动炮塔时摄像机跟随你的速度。

要增加或减小速度，进入战斗，使用 Ctrl + Shift + K 组合键打开控制台，并输入以下命令：vars.set cam_maxmove 3，其中 3 是所需的速度。

默认数字是 5。增加此值会使摄像机移动得更快，而减小该值会使其移动得更慢。


注意！目前，设置中的任何更改仅在下一次服务器更改前有效，之后它们将重置为默认值，需要再次更改。


指南

这种“无模式观察者”可用于审美或战略目的。如果想在没有官方观察者账户的情况下拍摄一些很酷的照片，请参考本指南。

然而，通过调整摄像机设置，你可以利用修改后的摄像机速度在战斗中获得优势。以马格南为例，你只需前往能让你最好地观察敌方基地的位置，并在任务选项卡中输入代码“vars.set cam_maxmove 0”。接下来是困难的部分；你需要开走并找一个好的地方躲起来。一旦你确定了你离摄像机旋转的地方有多远，你就可以向毫无防备的敌人开火了！

以下是在失落神庙上使用此功能的一个很好的例子：

步骤 图片 说明
1. 开到下方所示位置并停在相同位置。
2. 将你的摄像机提升到坦克上方的最高位置。
3. — 将摄像机速度设置为 0，并使用默认的 Q 键将摄像机倾斜回水平位置。
4. 找到一个合适的角度并进行一些练习射击，看看你的射击落在哪里。
5. 射击基地里的另一支队伍。

请记住，使用此技术，如果你开始受到攻击，你将变得脆弱，因为你看不到自己的坦克或生命值。为了有效地使用它，建议练习输入命令，或者准备好粘贴它，以便在控制台菜单中花费尽可能少的时间。
