无模式观察者设置指南

无模式观察者是一种修改后的观察者模式，在此更新后，普通玩家也可以使用。它允许玩家通过控制台命令调整摄像机行为，从而创建不寻常的摄像机角度、截屏或尝试不同的观看设置。

本文具有历史意义。此处描述的设置不再适用于游戏的新 HTML5 版本，但它们作为旧系统运作方式及玩家使用方式的记录仍然很有用。

设置无模式观察者

要使用旧的无模式观察者设置，玩家必须进入一场战斗，定位摄像机，并使用控制摄像机移动的控制台命令。

步骤 说明
1. 进入一场战斗。
2. 找到您想要用于图片或摄像机角度的位置。
3. 按 Ctrl + Shift + K 打开控制台。
4. 输入 vars.set cam_maxmove 0 并按 Enter。
5. 按 Ctrl + Shift + K 或 Escape 关闭控制台。
6. 驾驶坦克离开您准备好的屏幕或角度。

调整垂直摄像机速度

命令

vars.set cam_vspeed 0.5

此命令更改了使用 Q 和 E，或 PageUp 和 PageDown 时摄像机垂直移动的速度。

要更改该值，玩家需进入战斗，使用 Ctrl + Shift + K 打开控制台，并输入命令：vars.set cam_vspeed 0.5。末尾的数字控制所需的速度。

默认值为 0.7。增加该值会使摄像机移动得更快，而减小该值会使摄像机移动得更慢。


注意！当时，设置的更改仅保存到下一次服务器变更之前。之后，它们会重置为默认值，必须重新更改。


调整水平摄像机速度

命令

vars.set cam_maxmove 3

此命令更改了炮塔转动时摄像机跟随的速度。

要调整该值，玩家需进入战斗，使用 Ctrl + Shift + K 打开控制台，并输入命令：vars.set cam_maxmove 3。末尾的数字控制所需的速度。

默认值为 5。增加该值会使摄像机跟随得更快，而减小该值会使摄像机跟随得更慢。


注意！当时，设置的更改仅保存到下一次服务器变更之前。之后，它们会重置为默认值，必须重新更改。


用法

无模式观察者可用于审美目的，例如在没有官方观察者账号的情况下进行截屏。有关截屏用法的更多信息，玩家可以参考此指南。

一些玩家还尝试在战斗中进行摄像机设置。例如，使用马格南时，玩家可以将摄像机对准一个有用的观察角度，将水平摄像机移动设置为零，将坦克移开，然后在隐藏位置开火，同时使用准备好的摄像机角度。

这需要练习，因为玩家不会直接看着自己的坦克，如果受到攻击，可能会变得脆弱。

失落神庙示例

以下示例展示了此功能如何在失落神庙中使用。

步骤 图片 说明
1. 驾驶到所示位置并停在相同位置。
2. 将摄像机提升到坦克上方的最高位置。
3. — 将摄像机速度设置为 0，然后使用默认的 Q 将摄像机倾斜回水平位置。
4. 找到合适的角度并进行练习射击，看看您的射击落点。
5. 从准备好的角度向敌方基地开火。

实用说明

使用此技术有明显的缺点。由于摄像机不再正常居中于玩家的坦克，因此更难注意到靠近的敌人、追踪生命值并对危险做出反应。

为了高效使用该设置，玩家通常会练习快速输入命令，或将命令准备好粘贴到控制台中。这减少了在控制台菜单中花费的时间，并使返回战斗变得更容易。

结论

无模式观察者设置是旧版 3D 坦克的一个不寻常部分。它们允许玩家调整摄像机移动、创建特殊的观看角度，并尝试截屏或战斗视角。

虽然这些设置不再适用于游戏的新 HTML5 版本，但它们仍然是游戏历史的一部分，并展示了玩家如何创造性地使用旧的摄像机系统。
