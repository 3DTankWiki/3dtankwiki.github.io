关于摩天大楼

地图预览
类别 详情
概览 摩天大楼是一张相当大的地图，更重要的是，它位于高耸的摩天大楼之上，因此得名。它的大部分区域几乎没有掩体，而几乎所有的坡道都非常薄，并且在坡道旁边有危险的转弯，使其成为地形最困难的地图之一。期待在每个角落都有驻守者、镭射炮之类的人，别忘了，小心地雷！
战斗模式 死亡混战，团队对战，互夺军旗，控制点
规模 10 对 10，20 名玩家

事实

事实 信息
曾用名 摩天大楼最初被称为深渊，但在 2014 年 6 月 3 日发生了变化。

模式 策略
死亡混战

驻守战术
如果你是一个重型驻守者，摩天大楼是完美的地图。

镭射炮、激光炮、马格南和其他狙击炮塔在这里茁壮成长，这主要是由于两个主要原因：

1.这张地图有很多开放区域，这使得找到一个隐蔽的驻守者变得不可能。

2.有太多的驻守点。

例如，理想的驻守点是在摩天大楼的顶部，位于蓝队基地，配备镭射炮 / 泰坦或马格南 / 维京等强力装备。

如果你打算去那里，确保手头有大量的地雷，因为你需要通过埋设地雷来封锁通往你位置的坡道。

这将确保在你忙于射击其他人时没有人会溜到你身后。

其他驻守点包括地图的角落，这些地方几乎从不被驾驶，并提供大量的掩体。

然而，你需要小心不要从这些区域掉下去，因为它们中的大多数只建立在一排道具上。

潜行与窃取战术
这是一种战术，你必须使用正确的装备，预测你应该瞄准哪些敌人。

这在摩天大楼中几乎完美运作。

首先，你需要选择正确的装备。我建议使用既通用又致命的东西，比如离子炮 / 维京或火龙珠 / 猎人中甲。

然后，跳到中间较低的坡道之一，射击任何从你上方经过的人。

如果人们停止从那里经过，只需改变你的位置，但要安静地做！

团队对战

在团队对战中，最大的目标是击杀而不被击杀。

这意味着你的团队中应该有足够的磁力炮来补充哪怕是最轻微的生命值损失。

大多数人倾向于在这种格式中全力以赴，试图尽早且快速地结束游戏；但这并不总是有效。

赢得摩天大楼团队对战需要三样基本东西：掩体、比敌人更大的伤害输出和磁力炮。

虽然，和死亡混战一样，建议使用更高效的装备之一。

这里最好的战术是拿一个远程炮塔，装备一个伤害提升并射击任何穿过你路径的人！

互夺军旗

防守
在摩天大楼中防守你的基地是更容易完成的任务之一。

在这张地图中防守基地只有三种最流行的方法：

1.在交通最糟糕的地方埋设地雷。

2.让基地里的狙击手不断射击对方团队。

3.坚不可摧的三人组的正常防守策略，例如维京 / 冰风暴，猛犸象 / 轰天炮和猎人中甲 / 磁力炮。

尽管如此，下图显示了你的防守者应该理想地放置在哪里。

褪色的区域显示了大多数防守者正在操作的地方，而点是至少应该有一名防守者在场的地方。

较暗的点指的是使用重型组合的人，例如镭射炮 / 泰坦或马格南 / 猛犸象，而较浅的点可能是由临时防守者填补的地方。

进攻
在这张地图中进攻可能会有点棘手，红队可能比对面更难。

为了表现出色并为你的团队带来胜利，进攻者必须比敌人的防守打得更努力。

虽然可能，如果进攻缺乏力量，这通常意味着敌人将开始更努力地进攻，而你团队中的玩家可能不得不改变他们的角色，这会导致失败。

为了避免这一切，解决方案来自规划进攻、路线和正确的时机。当狙击手在重新装弹时，冲！

在进攻性防守者和支援者的支持下，进攻者可以开始影响并带回旗帜以进行安全占领。

装备精良的进攻者应该拥有一个快速且健康的底盘，并配备中远程炮塔。

控制点

要在摩天大楼控制点中获得成功，团队必须总共占领五个点，这些点分布在巨大的地图上。

A 点位于蓝队基地的正中间。

B 点位于它正前方，靠近那座巨大的砖红色建筑。

C 点位于最前方，就在同一类型的建筑旁边，上面有一个广告牌。

D 点位于红队基地。

E 点位于地图中间。

谈到战术，通常建议占领离基地最近的点，所以如果你的团队这样做并没有错。

团队中的一名玩家，离点位最近的那个人，应该占领基地里的点位并留在那里防守它。

由于 A、B 和 D、E 点位于两个基地旁边，两个团队在占领它们时都不应该有任何问题。

然而，C 点稍微偏向红队，这就是蓝队经常面临占领困难的原因。

然而，如果蓝队设法占领了 C 点，它就可以掌握整个红队的命运，所以强烈建议不要让蓝队占领 C 点。

摩天大楼控制点战斗的优秀玩家可以装备任何射程的炮塔，具体取决于底盘，底盘应该足够健康和稳定，以便尽可能长时间地留在周围，让点位保持他团队的颜色。

理想情况下，更适合前进以占领更多点位的玩家应该穿戴中远程炮塔以及轻中型底盘，例如激光炮 / 黄蜂轻甲，火龙珠 / 蜂王或雷暴炮 / 维京。

然而，喜欢防守点位的玩家需要中重型底盘以及致命的炮塔，例如极速炮 / 泰坦，火箭炮 / 维京或磁力炮 / 猎人中甲。
