游戏无法加载

1. 我看到灰色屏幕，之后没有任何反应：

1. 停止所有下载。不要忘记关闭所有种子客户端。
2. 将 http://tankionline.com 添加到你的浏览器和防火墙的白名单中，或者暂时禁用它们。
3.理 Flash 共享对象。
4. 清除浏览器/客户端缓存。

2. 出现“服务器关闭连接”错误：

1. 如果此错误频繁出现，请更改你的密码和电子邮箱地址。可能是其他人正在登录你的账号。
2. 清理 Flash 共享对象。
3. 尝试使用其他浏览器。浏览器可以从官方网站下载 —— Chrome、Firefox、Opera 或 Internet Explorer。
4. 可能是你的互联网连接速度太慢。尝试重置你的互联网连接（关闭路由器后再重新开启），并停止所有下载。
5. 尝试使用独立版 Flash 播放器进行游戏。

3. 出现“连接服务器失败”错误：

1. 清理 Flash 共享对象。
2. 检查你的路由器设置（关闭的端口），可能 5221、5222、5190 端口被封锁了。你需要打开其中一个端口。这可以由你的系统管理员或互联网服务提供商 (ISP) 的支持人员完成。
3. 错误也可能是由你的互联网连接问题引起的。尝试稍后再进入游戏。

4. 出现“2048”错误：

如果错误持续发生且你无法进入游戏， 1. 重置你的路由器，然后重启你的电脑。
2. 清理 Flash 共享对象。
如果问题偶尔发生，很可能是由于你的互联网连接不稳定导致的。

输入账号密码后出现错误

1. 输入后提示“密码错误”：

1. 你的登录名和密码必须以英文输入（区分大小写）。
2. 确保你的键盘已切换到英文输入法。
3. 禁用大写锁定 (Caps Lock) 模式，因为它在输入登录名和密码时经常会导致问题。
4. 如果你之前已将电子邮箱地址绑定到你的账号，请点击“丢失昵称或密码”链接并按照说明操作。
5. 如果你确定输入的信息正确但仍无法访问你的账号，可能是有人进入了你的账号。你需要联系技术支持 help@tankionline.com。如何写信给 help@tankionline.com。

2. 尝试进入游戏时提示我被封禁：

1. 如果你严重违反了游戏规则，你的账号将无法解封。你可以在这里阅读规则。
2. 如果你确定没有违反任何规则，那么可能是其他人进入了你的账号。在这种情况下，你需要联系技术支持 help@tankionline.com。如何写信给 help@tankionline.com。
