玩家评价
优点
一款出色的防御和支援武器。可以对多个敌人或视野外的敌人造成范围伤害。它拥有特殊的导弹模式，配备制导导弹。
缺点
在齐射模式下，在导弹发射前，你必须保持对目标的锁定。导弹的范围伤害可能会造成巨大的自伤。

装备改造

主条目： 装备改造

火箭炮的装备改造



肾上腺素 245 000

当坦克严重受损时增加炮塔伤害。 为增强伤害调节器添加了一个备用电池，仅在坦克底盘严重受损的极端情况下激活。从两个能源为伤害调节器供电是不安全的——它可能会爆炸。要求仅在严重受损时激活，可能是为了避免保修索赔。
优点
  • 普通和暴击伤害：+25%
缺点
  • 伤害加成仅在生命值 ≤25% 时激活
  • 当坦克受到冷冻状态效果时，效果被禁用。



远程火箭炸药 245 000

火箭配备了非接触式触发器和无线电发射器。在火箭飞行过程中双击射击键，将立即引爆它。单发射击时请小心！
优点
  • 双击射击键即可引爆火箭
  • 最小火箭速度：20 m/s
缺点
  • 火箭加速时间：1.5 s
  • 如果不小心，可能会造成自伤
  • 水平自动瞄准角度：±0°


导弹发射器 «猎人» 245 000

专为高精度射击爱好者设计的改装。瞄准时提供几乎瞬间的目标锁定，但每次发射只有一枚火箭。
优点
  • 瞄准时间：-75%
  • 齐射重装时间：-40%
缺点
  • 每次齐射火箭数：1
  • 瞄准恢复时间：-30%


导弹发射器 «旋风» 245 000

专为猎杀重型坦克而设计的改装。每次发射的火箭数量翻倍，但在瞄准时目标锁定时间也会增加。
优点
  • 每次齐射火箭数：8
  • 齐射火箭间隙：-40%
  • 瞄准恢复时间：+50%
缺点
  • 瞄准时间：+20%


燃烧导弹 4 990

暴击伤害会在短时间内使目标着火。 一种带有特殊火箭转换器的改装，在暴击时启用。将标准导弹转换为燃烧导弹，让你能将对手吞没在火焰中，在一段时间内造成持续的每秒伤害。
优点
  • 暴击会使所有受损敌方坦克的温度升高 +0.4 着火
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%


冷冻导弹 4 990

暴击伤害会在短时间内对敌方坦克施加«冷冻»状态效果。 一种带有特殊火箭转换器的改装，在暴击时启用。将标准导弹转换为冷冻导弹。让你能减缓敌人在战场上的移动速度，并在一段时间内几乎将他们的炮塔焊死在底盘上。
优点
  • 暴击会使所有受损敌方坦克的温度降低 1. 冷冻
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%


磁性导弹 4 990

暴击伤害会在短时间内对敌方坦克施加«电磁脉冲»状态效果。 一种带有特殊火箭转换器的改装，在暴击时启用。将标准导弹转换为电磁导弹。导致敌方坦克的电子设备短暂关闭，在一段时间内停用坦克的道具。
优点
  • 暴击命中敌人将对其施加 电磁脉冲状态效果，持续 3 秒。
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%


眩晕导弹 4 990

一种用于使危险目标瘫痪的改装。任何受到暴击伤害的坦克将在一段时间内无法移动。
优点
  • 暴击命中敌人将对其施加 眩晕状态效果，持续 1.5 秒。
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%


穿甲导弹 4 990

专为想要摧毁强大目标的赏金猎人设计的装备改造。任何受到暴击伤害的坦克将在一段时间内无法使用护甲。
优点
  • 暴击命中敌人将对其施加 穿甲状态效果，持续 9 秒。
  • 暴击几率：+10%
缺点
  • 暴击伤害：-50%


干扰导弹 4 990

伤害会在短时间内对对手施加干扰器状态效果。
导弹配备了铝涂层玻璃纤维。命中时会产生一小团云，使敌人的过速装置失效。
优点
  • 暴击命中敌人将对其施加 干扰器状态效果，持续 5 秒。
缺点
  • 暴击伤害：-50%


导弹发射器 «镭射»

伤害增加，但导弹加速缓慢。
一种带有加重导弹的火箭炮改装。为了确保发射器安全，这些导弹重装和加速缓慢，即使在齐射时也以最小速度离开炮管。命中目标感觉就像一场庆典，不仅因为这极其困难，还因为敌方坦克会因火箭载荷的巨大重量而爆炸成烟花。
优点
  • 伤害：+125%
  • 暴击伤害：+160%
  • 瞄准时间：-25%
  • 最小溅射伤害半径：+50%
  • 最大弹丸速度：350 m/s
缺点
  • 弹丸加速时间：+700%
  • 单发重装时间：+15%
  • 齐射火箭间隙：+400%


导弹发射器 «串联»

在一次齐射中发射两枚快速瞄准的火箭。火箭角速度、暴击伤害几率和齐射后重装时间增加。
遇到一个能预测每一个动作并躲避你所有火箭的对手？向他们发射一枚快速制导火箭！对手太酷了，连那个也躲过了？同时发射两枚快速制导火箭！看看他们怎么躲。
优点
  • 重装时间：-12%
  • 瞄准时间：-75%
  • 火箭角速度：+25%
  • 暴击几率：+25%
  • 齐射火箭间隙：-100%
缺点
  • 齐射后重装时间：+35%
  • 每次齐射火箭数：2


导弹发射器 «九头蛇»

一种速射火箭发射器。齐射会发射大量伤害降低且不可控的火箭。 这些火箭曾经安装在武装直升机上。由于缺乏瞄准系统且尺寸较小，可以在发射器中塞入多枚火箭。齐射攻击的持续时间长，允许在射击时在敌人之间切换，而不会中断齐射火箭的流动。当然，坦克手也没得选。
优点
  • 每次齐射火箭数 = 20
  • 齐射火箭间隙：-20%
  • 瞄准时间：-50%
  • 单发重装时间：-40%
缺点
  • 普通和暴击伤害：-50%
  • 齐射后重装时间：+80%
  • 火箭角速度 = 3°/秒


导弹发射器 «浮士德»

导弹射程缩短，但瞄准目标和重装所需时间更短。 就像剧院舞台上的艺术家一样，使用此装备改造的玩家为观众表演。没有幕间休息，战场上只有闪电般的事件过程。然而，这位艺术家是个新手，有时他们无法将自己的想法传达给剧院包厢里的观众。
优点
  • 重装时间：-50%
  • 瞄准时间：-50%
  • 齐射火箭间隙：-50%
缺点
  • 射程 = 40m
  • 齐射后重装时间：+100%


真空导弹

目标距离爆炸中心越远，受到的伤害越大。再次按下射击键可立即在空中引爆导弹。 远程和真空爆炸技术的独特结合，快速飞行的导弹如果瞄准目标附近而不是直接瞄准目标，会更有效。爆炸导弹产生的冲击波比直接命中造成的伤害更大。
优点
  • 平均和最小溅射伤害：235%
  • 双击射击键即可引爆火箭
  • 暴击伤害：+25%
  • 最小火箭速度：20 m/s
  • 水平自动瞄准角度：±0°
缺点
  • 伤害：-30%
  • 火箭加速时间：1.5 s
  • 如果不小心，可能会造成自伤
  • 齐射火箭间隙：+6%


导弹发射器 «流星»

命中目标时伤害增加。溅射伤害大幅降低，但冲击力增加。 感受流星的力量！你的敌人几乎没有机会在这次流星打击中幸存。然而，要利用这种粉碎性的力量，你必须仔细瞄准。专为真正的导弹专家设计的装备改造！
优点
  • 伤害：+25%
  • 冲击力：+50%
  • 最小火箭速度 = 90 m/s
  • 齐射火箭间隙 = 50 ms
缺点
  • 溅射半径 = 1m


导弹发射器 «铜指虎»

加速火箭齐射瞄准，但显著增加发射后的重装时间。启用火箭无线电引爆。
此改装是在分析了无数火箭炮战斗场景后开发的。增强的炮塔护甲确保发射器始终装满火箭，从而实现快速齐射部署。然而，齐射后的新鲜火箭重装需要更长时间。
应机密请求，发射器配备了用于飞行中火箭引爆的无线电引爆模块。此功能的详细信息仍然是绝密。
优点
  • 双击射击键即可引爆火箭
  • 重装时间：-12%
  • 瞄准时间：-62%
  • 齐射火箭间隙：-20%
  • 最大暴击几率 = 100%
  • 最小火箭速度：20 m/s
缺点
  • 齐射后重装时间：+111%
  • 火箭加速时间：1.5 s


露营者

当坦克受损最小时增加炮塔伤害。
增强伤害调节器的备用电池，通过人工智能控制的管理系统与主电源并联。该系统安全地将炮塔伤害提升至调节器的标准能力之外，但前提是坦克的结构完整性保持不变。如果人工智能检测到底盘受损，备用电池将立即关闭——当然，这一切都是为了防止保修索赔。
优点
  • 普通和暴击伤害增加：+90%
缺点
  • 伤害加成仅在生命值至少为最大生命值的 85% 时激活。
  • 当坦克受到 冷冻状态效果时，效果被禁用。


脉冲星


一种用不可复制的技术制造的装备改造。暴击会产生一个量子涡流，同时施加多种效果，通常是负面效果。科学家们设法降低了暴击伤害，但这不足以挽救局面——一旦被击中，敌人仍将惨败。该技术的发明者在国际通缉名单上。
优点
  • 暴击命中敌人将施加以下状态效果：
    • 干扰器：5 秒
    • 电磁脉冲：3 秒
    • 眩晕：1.5 秒
    • 穿甲：9 秒
缺点
  • 暴击几率：-9%
  • 暴击伤害：-90%


卓越


增强关键炮塔或底盘参数，而不引入新功能。
这种高级装备改造是突破性研究计划的成果，将钻石晶体集成到炮塔设计中。金刚石涂层的枪管内衬增加了射击时的操作压力。金刚石轴承提高了炮塔的旋转效率。金刚石半导体提高了瞄准系统的计算性能。唯一的缺点？其惊人的工艺隐藏在炮塔内部，无法从外部看到。
优点
  • 伤害：+30%
  • 转弯速度：+15%
  • 转弯加速度：+15%


凤凰 5 000 000

只有那些像凤凰一样一次又一次从灰烬中崛起以实现目标的坦克手才配得上这种武器。一种秘密的装备改造。使用火箭炮炮塔赚取 5 000 000 经验值即可获得。
优点
  • 伤害：+15%
  • 暴击伤害：+15%
  • 暴击几率：+30%
  • 垂直自动瞄准：+15%
  • 战斗中获得的经验值：+40%

特性表
  • 伤害 (hp) — 单次射击对目标造成的伤害。
  • 齐射伤害 (hp) — 如果所有弹丸命中，单次齐射对目标造成的伤害。
  • 暴击伤害 (hp) — 单次射击对目标造成的伤害。忽略除特殊“犰狳”模块外的任何防护模块。
  • 暴击几率 (%) — 对敌人造成暴击伤害的概率。
  • 冲击力 (条件单位) — 对目标造成的物理冲击，将坦克从其位置推开。
  • 爆炸冲击力 (条件单位) — 对弹丸爆炸半径内的坦克造成的物理冲击，将坦克从其位置推开。
  • 后坐力 (条件单位) — 射击时炮塔对自身底盘造成的物理冲击。
  • 重装 (s) — 无瞄准时的单发间隔。
  • 齐射后重装 (s) — 炮塔发射齐射后重装的时间段。
  • 瞄准时间 (s) — 将激光保持在目标上以完全锁定目标所需的时间。
  • 转弯速度 (deg/s) — 炮塔转动的最大速度。
  • 转弯加速度 (deg/s^2) — 炮塔加速到其最大转弯速度的加速度。
  • 射程 (m) — 弹丸爆炸前可飞行的最大距离。
  • 最大弹丸速度 (m/s) — 弹丸飞行期间可达到的最大速度。
  • 最小弹丸速度 (m/s) — 刚射出弹丸的速度。弹丸会加速直到达到“最大弹丸速度”值。
  • 弹丸加速时间 (s) — 弹丸达到最大速度所需的时间。
  • 火箭角速度 — 如果目标正在移动，制导火箭偏离先前航向的速度。
  • 齐射火箭数 — 单次齐射发射的火箭数量。
  • 齐射火箭间隙 (s) — 齐射中火箭发射之间经过的时间。
  • 瞄准恢复时间 (s) — 在不丢失任何进度的情况下可以恢复对目标瞄准的时间段。
  • 弹丸半径 (m) — 从弹丸中心到其末端的距离。如果弹丸飞过目标的距离不超过此距离，则会造成伤害。
  • 最大爆炸伤害半径 (m) — 爆炸造成全部伤害的范围。如果到目标的距离更大，伤害会降低至最小爆炸伤害范围。
  • 中等爆炸伤害半径 (m) — 图表断裂的一个额外点，随着爆炸点和目标之间距离的增加，爆炸伤害会降低。使伤害对距离的依赖性更呈指数级而非线性。
  • 最小爆炸伤害半径 (m) — 爆炸仍能造成伤害的范围。如果从爆炸点到目标的距离更大，则不会造成伤害。
  • 暴击爆炸伤害半径 (m) — 爆炸造成暴击伤害的范围。如果到目标的距离更大，则根据特性造成普通伤害。
  • 中等爆炸伤害百分比 (m) — 在中等爆炸伤害距离处对目标造成的伤害百分比。
  • 向上自动瞄准 (deg) — 高于炮塔瞄准平面的目标自动瞄准角度。
  • 向下自动瞄准 (deg) — 低于炮塔瞄准平面的目标自动瞄准角度。
