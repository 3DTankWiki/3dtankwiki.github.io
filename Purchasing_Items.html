在商店购买物品

在商店购买物品有几种方式。有关每种方式的更多信息，请查看以下部分。

使用坦克币
  1. 在大厅界面，点击账号面板（显示你的等级、水晶等的栏）上的商店按钮

  2. 向下滚动查看优惠 - 并非所有物品都可以使用坦克币购买。例外情况是礼包、VIP 和坦克币本身。在尝试购买之前，请确保你有足够的坦克币。商店中有一个坦克币部分，你可以用真钱购买坦克币。


  3. 选择一个物品并点击绿色的 CONFIRM [N] 按钮


  4. 如果你有足够的资金且一切顺利，你应该会看到如下界面。购买所使用的坦克币也将从你的坦克币总余额中扣除。


通过 Paypal 购买
  1. 前往商店，通过选择你想要购买的任何礼包进入支付页面。
  2. 点击 PayPal 按钮


  3. 点击 «Purchase for real money» 按钮。


    (你将被重定向到 Paypal 的支付页面)
  4. 登录你的 PayPal 账号
    1. 输入你的详细信息
    2. 点击“继续”
    3. 如果你没有账号，请注册并将信用卡绑定到你的账号


    4. 如果有任何其他额外步骤，只需按照提示操作，点击“下一步”/“确认”或类似按钮
  5. 如果你操作正确，你购买的商品很快就会出现在你的账号中（账号首次 PayPal 购买可能会有长达 24 小时的延迟），一旦发生这种情况，你将看到此窗口：


通过信用卡购买
  1. 前往商店，通过选择你想要购买的任何礼包进入支付页面。
  2. 点击信用卡按钮。


  3. 点击 «Purchase for real money» 按钮。


  4. 输入你的卡号、有效期、持卡人姓名和 CVV2/CVC2 码。
  5. 点击“购买”以确认支付。


如果有任何其他额外步骤，只需按照提示操作，点击“下一步”/“确认”或类似按钮。 如果你操作正确，你购买的商品很快就会出现在你的账号中，一旦发生这种情况，你将看到此窗口：


通过 Paysafecard 购买
  1. 前往商店，通过选择你想要购买的任何礼包进入支付页面。
  2. 点击 "PaysafeCard"
  3. 点击 «Purchase for real money» 按钮。


  4. 从那里，它应该会重定向到 Paysafecard 的官方网站。
    1. 填写必要信息
    2. 点击“支付”以确认支付


你购买的商品很快就会到达游戏中，你将看到如下界面：



请注意：PaySafe 并非在每个国家的商店都可用。美国商店没有 PaySafe 选项。

通过礼品卡购买
  1. 前往商店，通过选择你想要购买的任何礼包进入支付页面。
  2. 点击礼包并选择如下所示的礼品卡选项。


  3. 你将被重定向到浏览器中显示的如下页面。

    点击“选择你的品牌”下拉菜单，选择你拥有的礼品卡品牌。要查看接受哪些品牌的礼品卡，请点击这里。
  4. 在提供的空格中输入卡号和卡 PIN 码。


  5. 检查你的余额，如果足够，请继续下一步。但请记住，每张礼品卡只能使用一次！


如果一切顺利，你应该会收到你的物品：


通过 Openbucks 购买
  1. 前往商店，通过选择你想要购买的任何礼包进入支付页面。
  2. 根据你拥有的礼品卡类型，点击 CVS、Dollar General 或 Obucks 按钮。
  3. 点击 «Purchase for real money» 按钮。
  4. 输入你的礼品卡号和 PIN 码。勾选确认你已阅读并同意所述使用条款的方框。点击“支付”按钮。

如果你正确完成了这些步骤，购买的物品很快就会出现在你的账号中。

如果你没有看到此类支付选项或在完成支付时遇到任何问题，请确保你的账号设置中的位置为“美国”，并且在完成支付时没有使用任何 VPN 服务。

支付交易失败

如果你支付了钱但没有收到水晶、VIP 或金箱子道具，该怎么办。

不幸的是，有时你支付了商品（水晶、VIP、金箱子道具）但它们没有出现在你的账号中。这通常是由于支付提供商的问题导致的。 但别担心！你的钱没有丢失，在大多数情况下，这个问题可以很快解决。

在报告问题之前，请等待几个小时，因为支付可能需要一些时间（特别是如果你使用短信支付）。

如果你的水晶仍然没有出现，只需填写此表格并提交。


尽可能详细但简洁地描述问题。更好的描述将导致问题处理得更快。
