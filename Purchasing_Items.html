使用坦克币
  1. 在您的大厅屏幕上，点击账号面板上的商店按钮（您可以看到您的等级、水晶等的那条栏）

  2. 向下滚动以查看优惠 - 并非所有物品都可以使用坦克币购买。例外包括礼包、VIP账号和坦克币本身。在尝试购买之前，请确保您有足够的坦克币。有一个坦克币部分在商店里，您可以用真实货币购买坦克币。


  3. 选择一项并点击绿色的确认 [N] 按钮


  4. 如果你有足够的奖励，并且一切顺利，你应该会看到如下的屏幕。购买中使用的坦克币也将从你的总坦克币余额中扣除。


通过Paypal购买
  1. 去商店，选择你希望购买的任何套餐，然后到达支付页面。
  2. 点击PayPal按钮


  3. 点击«充值购买»按钮。


    (您将被重定向到PayPal的付款页面)
  4. 登录到您的 PayPal 账号
    1. 输入您的详细信息
    2. 点击‘继续’
    3. 如果您没有账号，请注册并将信用卡绑定到您的账号


    4. 如果还有其他额外步骤，请关注它们，点击‘下一步’/‘确认’或类似的按钮。
  5. 如果你一切都做对了，你购买的商品应该很快出现在你的账号中（第一次在账号上使用PayPal购买可能会有最长24小时的延迟），一旦发生这种情况，你将看到这个窗口:


通过信用卡购买
  1. 去商店，选择任何您想购买的套餐，然后到达支付页面。
  2. 点击信用卡按钮。


  3. 点击«充值购买»按钮。


  4. 输入您的卡号、到期日期、持卡人姓名和CVV2/CVC2代码。
  5. 点击“购买”以确认支付。


如果还有其他的额外步骤，请关注它们，按下‘下一步’/‘确认’或类似的按钮。如果你一切操作正确，你购买的商品应该很快就会出现在你的账号中，一旦发生这种情况，你将看到这个窗口:

  2. 点击 "PaysafeCard"
  3. 点击«充值购买»按钮。


  4. 从那里，它应该重定向到Paysafecard的官方网站。
    1. 填写必要的信息
    2. 点击“支付”以确认付款

通过 Openbucks 购买
  1. 去商店，选择你想购买的任何套餐进入支付页面。
  2. 根据您拥有的礼物卡类型，点击CVS、Dollar上将或Obucks按钮。
  3. 点击«充值购买»按钮。
  4. 输入您的礼物卡号码和其密码。检查框以确认您已阅读并同意上述使用条款。点击«支付»按钮。

如果您正确完成了步骤，购买的商品应该很快会出现在您的账号上。

如果您没有看到这样的支付选项，或者在完成支付时遇到任何问题，请确保在您账号的设置中将«USA»作为您的位置，并且在完成支付时不使用任何VPN服务。

支付交易失败

如果您支付了钱却没有获得水晶、VIP账号或金箱子道具，请按照以下步骤操作。

不幸的是，有时你为商品（水晶，vip 账号，金箱子道具）付费，但它们并没有出现在你的账号中。这通常是由于支付提供商出现问题造成的。但别担心！在大多数情况下，你没有失去你的钱，这个问题可以迅速解决。

在报告问题之前，支付后请等待几个小时，因为付款可能需要一些时间（特别是如果您使用短信支付的话）。
